## 三、第二部分：小游戏设计与实现

### 1. 已实现小游戏 1：墓门验证

所在仓库：

```text
/Users/fanjiaqi/Documents/小游戏/aiag
```

涉及文件：

```text
index.html
styles.css
game.js
```

玩法流程：

```text
墓门场景收集线索
→ 点击门洞深处
→ 弹出“墓门验证”小游戏
→ 按顺序点击：额 → 砂 → 雾 → 门
→ 成功后进入甬道
```

对应主线线索：

```text
额：门额背面彩画
砂：封门砖缝石英砂
雾：墓道墙面水汽
门：门洞深处
```

设计目的：

```text
把普通点击收集线索转化为一个主线验证动作，让玩家不是直接进入下一场景，而是先整理墓门区域的信息，再完成墓门通行判断。
```


```text

```
