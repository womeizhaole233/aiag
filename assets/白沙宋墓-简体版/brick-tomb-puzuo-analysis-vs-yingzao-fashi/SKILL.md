# 砖墓铺作与《营造法式》差异分析

## 适用对象
- 墓门铺作
- 前室铺作
- 后室铺作

## 使用前提
- 已知铺作构件名称（如华棋、昂、令棋等）
- 掌握《营造法式》相关条文

## 执行步骤

1. **确认铺作基本形制**：记录铺作类型（如单抄单昂重棋五铺作）及各跳出跳长度。
2. **比对关键差异点**：
   - 检查棋端卷杀瓣数是否符合《营造法式》（华棋、泥道棋应为四瓣，令棋应为五瓣）。
   - 测量泥道棋与令棋长度，判断是否违反“泥道棋短于令棋”的规定。
   - 识别昂的形式（如琴面昂），并对照《营造法式》及同期木构实例判断其时代特征。
3. **核算材栔比例**：测量材高与栔高，计算比例并与《营造法式》标准（15:6）比较。
4. **评估砖材制约影响**：
   - 检查斗件高度是否为砖层整数倍（如牯斗耳、平、欹各高5厘米对应三层砖厚）。
   - 判断是否存在结构妥协（如偷心造华棋额外承托柱头方）。
5. **输出结论**：指出该墓铺作在卷杀形式、构件比例、昂制等方面偏离《营造法式》，反映砖材对仿木结构的制约。

> **注意**：本技能仅适用于砖砌仿木结构，不适用于真实木构建筑。
