| `song-dynasty-brick-chamber-tomb-excavation` | 宋代砖室墓发掘 | 执行带有彩绘的宋代砖室墓系统性抢救性发掘、记录与初步保护 |

## 二、墓葬结构与建筑分析

| 目录名（英文） | 中文翻译 | 含义说明 |
|---|---|---|
| `bai-sha-song-mu-spatial-layout-and-burial-structure` | 白沙宋墓空间布局与墓葬结构 | 分析墓葬整体朝向、五段式结构序列及砖室墓对地上木构建筑的象征性模仿 |
| `brick-tomb-puzuo-analysis-vs-yingzao-fashi` | 砖墓铺作分析与《营造法式》对比 | 分析砖砌仿木铺作形制，并与《营造法式》对比，识别材料限制导致的构造偏差 |
| `song-dynasty-brick-imitation-wood-puzuo-analysis` | 宋代砖仿木铺作分析 | 分析北宋砖砌仿木结构墓门与转角铺作的构件尺寸、组合方式及其与官式规范的关联 |
| `identify-luding-ceiling-in-architectural-contexts` | 建筑语境中的盝顶识别 | 识别墓室顶部或殿堂内部是否采用盝顶（平棋形式）及其历史定位 |
| `dou-liu-caisson-structural-analysis` | 斗六藻井结构分析 | 分析墓室顶部六角形藻井的构造细节、铺作形式及与辽宋实例的对应关系 |
| `shi-di-pattern-identification-in-song-liao-architecture` | 宋辽建筑中的柿蒂纹识别 | 识别四瓣花形纹样是否为柿蒂纹及其变体，判断其时代特征与地域归属 |
| `muzang-xiaomu-zuo-men-chuang-guijiao-fenxi` | 墓葬小木作门窗龟脚分析 | 分析墓室砖砌仿木门窗（版门、破子棂窗等）及龟脚构件的形制与《营造法式》对照 |
| `song-mu-xiao-mu-zuo-imitation-analysis` | 宋墓小木作模仿分析 | 综合判断白沙宋墓整体构造是否仿效《营造法式》小木作系统 |
| `mu-shi-pu-zuo-cai-deng-identification` | 墓式铺作材等识别 | 根据铺作材高对照《营造法式》材等标准，判断其所属木作系统（大木作或小木作） |
| `interpret-column-base-lotus-decoration-as-rafter-simulation` | 柱脚莲花装饰作为檩仿制的解读 | 解读柱脚单独绘制莲花纹是否为对地上建筑柱檩莲花彩画的象征性模仿 |
| `song-dynasty-brick-tomb-sealing-analysis` | 宋代砖墓封门分析 | 分析封门砖的层数、砌法及预留空隙，指导安全拆除并判断墓葬等级 |
| `zhuan-chuang-ao-zi-xing-ping-mian-yuan-liu-fen-xi` | 砖床"凹"字形平面源流分析 | 识别唐宋墓砖床"倒凹字形"平面形式，追溯其模仿的地上居室原型 |

## 三、壁画与装饰研究

| 目录名（英文） | 中文翻译 | 含义说明 |

| `bai-sha-song-mu-wall-painting-functional-analysis` | 白沙宋墓壁画功能分析 | 对一号墓壁画进行空间区域划分，基于服饰、动作、持物等判定人物身份与社会功能 |
| `song-dynasty-tomb-imitation-wood-painting-restoration` | 宋代墓葬仿木彩画复原 | 根据残存彩画痕迹和《营造法式》对影作仿木构件（阑额、斗拱等）彩画进行复原 |
| `qing-lv-die-yun-cai-hua-source-analysis` | 青绿叠晕彩画源流分析 | 分析建筑彩画中青绿叠晕棱间装与枋心彩画的历史源流、技法及演变 |
| `yingzao-fashi-song-dynasty-painting-classification` | 《营造法式》宋代彩画分类 | 根据纹样、位置、用色判断彩画是否属于《营造法式》"解绿结华装"制度的早期实践 |
| `baisha-song-murals-preservation-and-zoning` | 白沙宋墓壁画保存与分区 | 对二号墓因泥水渗透损毁的残存壁画进行保存状况评估，按甬道壁与墓室壁分区记录 |
