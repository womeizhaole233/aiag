# 山华蕉叶装饰传统分析

## 何时使用
- 墓室壁上端存在帐式装饰结构
- 石窟或佛龛采用盏顶帐形式
- 装饰构件疑似源自《营造法式》所载小木作制度
- 需判断某处蕉叶形雕饰是否属于山华蕉叶传统谱系

## 执行步骤
1. **确认结构前提**：验证对象是否具备“帐式结构”和“仿小木作装饰”两个先决条件。
2. **排除不适用场景**：若为无帐式顶部的普通墙体，则本技能不适用。
3. **比对历史范式**：参照南北朝至晚唐的盏顶帐实例（如云冈、龙门、敦煌石窟），检查装饰形式是否符合山华蕉叶的典型特征。
4. **引用《营造法式》规范**：
   - 若为佛道帐，检查是否“压厦版外出四十分，上施混肚方，方上用仰阳版，版上安山华蕉叶”
   - 若为牙脚帐，检查是否“铺作之上用压厦版，版上施混肚方、仰阳山华版”
5. **得出结论**：若装饰位置、形制与文献及实例一致，则确认该装饰直接继承佛道帐类小木作传统制度。

> 注意：本技能仅适用于具有明确帐式顶部结构的场景，不可用于普通墙面装饰分析。
