# 一桌二椅对坐布局商业起源分析

## 何时使用
- 在墓室壁画、古代绘画或考古复原场景中识别出“一桌二椅二人对坐”布局；
- 需判断该布局的社会文化来源；
- 明确排除家庭、官舍、寺观等非商业空间语境。

## 执行步骤
1. **确认布局特征**：验证是否存在两人相对而坐、共用一桌的明确空间模式。
2. **比对历史图像与文献**：
   - 对照《清明上河图》《雪山行旅图》《文姬归汉图》等宋代绘画中的饮食店铺布局；
   - 引用《东京梦华录》《归田录》等文献中关于酒楼“两人对坐饮酒”的记载。
3. **排除非商业场景**：
   - 检查是否出现在家庭、官舍或宗教场所——若出现，则本技能不适用；
   - 参考司马光《书仪》等礼制文献，确认士大夫家庭婚仪等场合仍采用分桌制，未采用对坐布局。
4. **得出推断**：若布局仅见于商业餐饮空间及相关衍生场景（如墓葬装饰），则可判定其起源于北宋城市饮食店铺，并通过商业文化传播至丧葬艺术。

> 注意：本技能仅适用于可明确辨识人数、方位及桌椅组合的场景。
