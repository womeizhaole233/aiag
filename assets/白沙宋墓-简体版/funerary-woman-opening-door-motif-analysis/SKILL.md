# 妇人启门装饰母题分析

## 触发条件
当装饰中出现以下特征时应用本技能：
- 存在假门或门形结构
- 门后有妇人或小童半身探出，作启门状
- 装饰位于墓室、石棺、塔幢、牌楼、香炉或铜镜等载体上

> 注意：不适用于无门形结构的纯平面壁画。

## 执行步骤

1. **确认载体与结构**：识别装饰所在对象是否属于适用范围（宋金元明清墓葬、石棺、塔幢、铜镜、牌楼、香炉）。
2. **判定时代与地域**：根据风格、铭文或考古背景，确定所属历史时期（东汉至民国）及地理区域（如四川、河北、山西、朝鲜等）。
3. **追溯演变脉络**：
   - 起源：东汉（如四川芦山王晖石棺、徐州汉画像石）
   - 盛行：北宋（白沙宋墓、郑州柿园宋墓等）
   - 发展：金代（格子门替代版门、增加手持衾盒等细节）
   - 传播：高丽初期铜镜纹饰
   - 衰歇与遗存：元明清至民国仍有零星使用
4. **解读象征意义**：该母题表达“假门之后尚有空间”的死后世界延续观念，反映墓主对幽冥居所的想象。
5. **关联文学与审美**：结合宋代词曲（如《玉照新志》《董西厢》）及画院作品，理解其与当时“妇女倚门”审美风尚的互动。
