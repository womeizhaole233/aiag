# 宋代买地券分析技能

## 适用条件
仅适用于满足以下全部前提的出土文物：
- 出土带文字的地券
- 券文为朱书（红字书写）
- 配有券盖

不适用于无文字或非买地性质的砖券。

## 触发判断
若墓中出土朱书砖券，且内容包含四至（东、西、南、北方位神兽）、钱数（如“九万九千九百九十九贯文”）及神祇名（如“丘丞墓伯”“道路将军”），则判定为标准宋代买地券，启动本技能。

## 执行步骤

1. **确认物理特征**
   - 检查地券材质是否为砖质
   - 测量尺寸（典型如长38cm、宽28cm、厚2.3cm）
   - 观察上端是否斜杀二角
   - 核实券盖是否存在，是否无文字、有压绳纹

2. **识别文字书写方式**
   - 确认券面为倒写朱书
   - 阅读方向应自左向右
   - 注意文字漫漶情况，优先辨识关键词

3. **解析标准文本结构**
   按顺序识别以下组成部分：
   - 开头：年月日、死者姓名、龟筮叶从、相地袭吉
   - 交易内容：含标准金额（通常为“九万九千九百九十九贯文”）及“五彩信币”
   - 四至界定：东至青龙，西至白虎，南至朱雀，北至玄武
   - 神祇系统：内方勾陈、丘丞墓伯、道路将军等
   - 誓约条款：如“千秋万岁，永无殃咎……收付河伯”
   - 交割确认语：“财地交相，各已分付……永保休吉”
   - 见证人信息：知见人、岁月、主保人
   - 驱邪咒语结尾：“急急如五帝使者女青律令”

4. **还原仪式使用流程**
   - 斩草日：丹书铁券（或砖券），埋于墓穴地心
   - 入葬时：另埋一券于棺柩前方
   - 验证两券背面是否书有“合同”字样，以确认契约成立

5. **输出结论**
   综合以上信息，判定该地券是否为模拟土地买卖契约，并说明其通过神灵见证确立墓主对阴宅合法所有权的功能。
