# 推断宋代无棺直葬方式

## 适用条件
- 墓室已清理完毕
- 人骨保存状态可辨识
- 不适用于存在棺椁痕迹或金属葬具残留的墓葬

## 执行步骤

1. **观察人骨位置与姿态**
   - 记录人骨在墓室中的具体方位（如头向、脚向、面朝方向）
   - 分析多具人骨之间的相对关系（如叠压、并置、倚靠）
   - 判断原始尸体摆放姿态是否符合直接安置特征

2. **检查葬具缺失证据**
   - 确认人骨四周无铁钉、金属构件等葬具附件
   - 检查周围土壤或砖面是否存在朽木痕迹
   - 测量人骨与墓壁的最小距离（壁距），若空间不足以容纳棺椁，则支持无棺推断

3. **综合判断葬式**
   - 若上述证据一致表明无葬具存在，且人骨直接置于砖床，则推断为无棺直葬
   - 结合随葬品情况（如无实物随葬但可能使用已焚化的纸明器）辅助判断时代与社会背景

4. **输出结论**
   - 明确提示该墓采用无棺直葬方式
   - 关联至北宋中晚期特定社会阶层或丧葬风尚

> **注意**：本技能专用于类似白沙宋墓第三号墓的考古情境，需严格排除任何葬具残留证据后方可应用。
