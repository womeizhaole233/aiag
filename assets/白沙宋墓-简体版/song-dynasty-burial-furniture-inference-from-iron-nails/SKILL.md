# 基于铁钉遗迹推断宋代合葬棺形制

## 何时使用
- 出土两具人骨，呈头西脚东、男北女南横陈于砖床
- 人骨周围发现排列成行的铁钉（非散乱状态）
- 未发现棺木实物，但需复原葬具形制
- 考古现场位于宋代中原地区（如白沙、叶县、登封等）

## 执行步骤
1. **观察人骨状态**：记录人骨数量、相对位置、朝向及与周围遗物的混合情况。
2. **记录铁钉信息**：统计铁钉数量、测量长度范围（通常为4–8厘米），并测绘其空间分布。
3. **推断棺木参数**：
   - 根据铁钉围合范围确定棺木数量（通常为1具合葬棺）
   - 测量铁钉分布东西向长度作为棺长（约1.75米）
   - 测量西端（棺头）铁钉间距作为棺头宽（约65厘米）
   - 测量东端（棺尾）铁钉间距作为棺尾宽（约43厘米）
4. **结合文献佐证**：检索《辍耕录》、地方志（如隆庆《登封县志》）等史料，验证夫妻同棺葬俗在该地区的存在可能性。
5. **输出结果**：明确说明推断所得的棺木数量、尺寸（长、头宽、尾宽）及葬制类型（如“夫妻合葬一棺”）。

> 注意：本方法仅依赖铁钉围合轮廓，若铁钉缺失、移位或无规律，则不可用。
