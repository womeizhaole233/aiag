# 《营造法式》宋代彩画制度分类

## 适用条件
- 彩画存在于宋代或接近宋代时期的遗迹（如白沙宋墓、敦煌宋初窟檐、南唐李昇墓等）
- 纹样可识别，且包含以下任一类型：叠胜、方胜、半方胜、四半方胜、柿蒂、牡丹、仰莲
- 构件位置属于墓室彩画常见部位：斗拱、柱、枋、棋眼壁、墓顶、甬道、前室等

## 执行步骤
1. **识别纹样类型与位置**：确认纹样种类及其所在建筑构件。
2. **对照《营造法式》卷十四与卷三十二**：
   - 叠胜：若见于甬道、前室、过道、顶心，对应“罗文叠胜”
   - 方胜类：若用于额、枋、铺作、墓顶，符合“方胜宜于方、枋、斗棋内相间用之”
   - 柿蒂：若用于柱头箍头、小斗、棋等，属“细锦花纹之一种”
   - 牡丹：若绘于棋眼壁，符合“牡丹花……棋眼壁皆可用之”
   - 仰莲：若画于大小斗斗欹处，对应“斗下莲华，并以青晕”
3. **分析用色特征**：检查是否采用赭（地色）、青（花色）、白（轮廓线）为主，间用淡黄、墨绿。
4. **判定制度归属**：若满足“朱地上间诸华”特征，则归为《营造法式》所载“解绿结华装”的早期形式。
5. **输出结论**：明确指出该彩画反映北宋中期彩画制度，并早于《营造法式》成书（1103年）。

## 注意事项
- 不适用于非宋代遗迹或纹样无法辨识的案例
- 若仅部分纹样匹配，需结合整体构图与用色综合判断
- 此技能聚焦制度归属，不涉及修复或复制工艺
