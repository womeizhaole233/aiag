# 青绿叠晕彩画源流分析

## 触发条件
当彩画呈现以下特征时启用本技能：
- 构件为小斗棋、枋心或梁栿
- 可辨识青线或色彩层次
- 存在退晕（由深至浅的色彩过渡）效果
- 非单色平涂彩画

## 执行步骤
1. **确认《营造法式》规范**：核查是否符合卷十四“青绿叠晕棱间装”——外楼用青叠晕，身内用绿叠晕，仅作两晕。
2. **比对白沙宋墓变体**：若彩画保留退晕结构但未使用青绿相叠，应视为宋代地方变体。
3. **分析枋心处理方式**：检查枋心正中是否施有一道青线，并推断其退晕做法是否与棱间装一致。
4. **追溯后世影响**：识别清代旋子彩画中“桃尖一字枋心”等元素，确认其与宋代制度的传承关系。
5. **考察早期渊源**：参考日本平等院凤凰堂（1053年）等同期遗存，验证该技法在北宋初期的成熟度。
6. **解释技术本质**：说明“叠晕”模拟织物晕染效果，关联唐代染织传统（如正仓院实物及《延喜式》“晕绸”记载）。
