# 白沙宋墓空间布局与仿木构建筑分析

## 适用条件
- 分析对象为白沙宋墓一号墓或其他北宋中原地区砖室墓
- 墓葬结构保存完整，具备测绘数据
- 墓室存在仿木构件（如铺作、影作等）

## 执行步骤

### 1. 确定墓葬朝向与结构序列
- 测量墓葬主轴线方位角：白沙宋墓一号墓为北偏东15度
- 自南向北依次识别六部分：墓道 → 墓门 → 甬道 → 前室 → 过道 → 后室
- 标注各部分功能：
  - 墓道：连接外部通道
  - 墓门：仿木构砖砌门楼，正式入口
  - 甬道：过渡空间
  - 前室：第一主室，具完整铺作与壁画
  - 过道：设窗与过梁门，连接前后室
  - 后室：六角形主葬室，设假门、砖床及“穴”孔

### 2. 解析仿木构建筑原理
- 确认墓室为砖材所建，无法实现木材的大跨度结构
- 识别两种适应性做法：
  - **影作**：在砖面雕刻或砌出木构件视觉效果
  - **简化铺作**：如一斗三升、把头绞项造等极简斗栱形式
- 判断其本质为对地上木构的象征性再现，而非完全复制

### 3. 关联礼制思想
- 将“前室—后室”结构对应“前堂后寝”的宋代居室礼制观念
- 结合仿木构件说明墓葬作为“地下宅第”的文化意图

## 输出结果
返回墓葬朝向、结构序列清单及仿木构造类型说明，用于支撑墓葬制度、建筑史或丧葬观念研究。
