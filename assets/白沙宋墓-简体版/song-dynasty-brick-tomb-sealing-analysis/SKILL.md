# 宋代砖室墓封门结构分析

## 适用条件
- 墓门已暴露且封门砖未被扰动
- 仅适用于砖室墓，不适用于木椁墓或石室墓

## 执行步骤
1. **确认外层结构**：自底部至门额高度，检查是否采用“横砖”与“菱角牙子”混合砌筑。
2. **分析内层结构**：
   - 计数“卧丁砖”（砖长边垂直于墙面）的行数，应为八行；
   - 确认其上是否有一行“横砖”（砖长边平行于墙面）；
   - 检查最上行卧丁砖中部是否预留空隙。
3. **记录关键参数**：
   - 测量预留空隙的长度与宽度（单位：厘米）；
   - 记录卧丁砖总行数。
4. **推断功能与意义**：结合空隙位置与尺寸，判断其可能用于工具插入或仪式性通道，辅助评估防盗设计与工艺精细化程度。

## 输出用途
提供封门砖构造参数，用于：
- 指导安全拆除操作
- 判断墓葬等级与时代特征
- 分析宋代砖作技术特点
