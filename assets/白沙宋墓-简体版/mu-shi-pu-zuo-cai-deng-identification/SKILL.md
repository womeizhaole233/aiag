# 墓室铺作材等识别

## 适用条件
- 存在明确的铺作结构
- 可测量铺作中栱断面高度（材高）
- 不适用于无明确材高的纯装饰性砌筑

## 执行步骤
1. **测量材高**：获取墓室铺作构件的材高数据（单位：厘米）。
2. **换算宋尺**：按1宋尺 ≈ 31厘米，将材高换算为宋寸。
3. **对照《营造法式》材等标准**：参考卷四《大木作制度·材》中的材等规格。
4. **判定归属系统**：
   - 若材高接近第八等材（广四寸五分，约14厘米），则指向小木作系统（如殿内藻井、小亭榭铺作）。
   - 若接近第七等材（广五寸二分五厘，约16.3厘米），则可能用于小殿及亭榭，但仍需结合使用场景判断。
5. **输出结论**：明确指出该墓室铺作仿制对象属于小木作还是大木作系统。

> 提示：本技能聚焦于通过材高这一核心参数进行制度归属判断，详细材等对照表与历史背景参见参考资料。
