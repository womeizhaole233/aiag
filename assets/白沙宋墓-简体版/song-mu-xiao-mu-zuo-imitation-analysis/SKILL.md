# 宋墓小木作仿制分析

## 何时使用
- 分析对象为白沙宋墓三墓整体或类似高规格宋代砖室墓
- 墓内出现多种精细装饰构件（如龟脚、山华蕉叶、斗六藻井等）
- 需判断该墓葬是仅模仿大木作结构，还是深入仿效《营造法式》中的小木作系统
- 排除仅局部仿木的简单墓葬

## 执行步骤
1. **综合观察五项关键证据**：
   - **龟脚制度**：确认是否仅见于《营造法式》小木作（如佛道帐、经藏）
   - **山华蕉叶**：判断是否源自盏顶帐等小木作传统装饰
   - **盝顶与斗六藻井**：核实是否属于殿堂内部小木作天花系统
   - **铺作材高**：测量是否接近第八等材（专用于“殿内藻井”）
   - **装饰繁缛程度**：评估是否超出一般木建筑范围，趋近小木作极致

2. **对比一般仿木砖室墓**：
   - 普通墓葬通常仅模仿大木作柱梁斗栱
   - 若三墓深入至帐、藏、藻井等细节，则指向小木作仿制

3. **得出结论**：
   - 若多项证据一致符合小木作特征，则判定其整体构造仿效小木作系统
   - 此结论应作为解释墓主身份、时代风尚及社会文化镜像的核心框架

4. **关联历史背景**：
   - 将结论与北宋寺观、商铺中小木作流行的史实对照，强化解释力
