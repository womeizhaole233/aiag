# 唐宋熔铁灌缝砌筑技术识别与应用

## 何时使用
- 在唐宋时期的砖石结构中观察到构件间异常牢固、难以仅靠砌筑稳定；
- 发现缝隙中有铁锈、不规则金属残留或疑似铁质填充物；
- 研究对象属于适用类型：过梁门、石桥、墓门或水窗。

## 如何执行
1. **初步判断**：检查结构是否具备以下特征：
   - 构件间存在厚度约5–7毫米的不规则铁片状残留；
   - 铁片呈锈蚀状态且与砖石紧密结合；
   - 结构整体性极强，尤其在承重部位。

2. **历史对照**：结合文献记载判断时代合理性：
   - 唐代实例：乾陵玄宫石门“铸铁以固其中”、赵州桥“熔铁灌其中”；
   - 宋代沿用：汴河石岸“以锡、铁灌其缝”、《营造法式》载水窗“以锡灌”。

3. **验证建议**：
   - 对疑似部位进行无损金属探测；
   - 必要时实施微损取样，检测铁质成分及铸造特征；
   - 排除预制铁件可能性——该技术所用铁片为现场浇铸，形制不规则。

4. **排除条件**：若结构无任何铁质证据或不属于唐宋时期典型应用场景，则不适用本技术推断。

> 注意：本技能仅适用于存在铁质粘连证据的唐宋砖石结构，不适用于普通无金属加固的砌体。
