# 宋元考古中的域外材料参照方法

当研究宋元器物且存在域外相似遗存时，执行以下步骤：

1. **确认域外材料来源**：主要参考高丽（对应北宋至元时期）与日本（奈良、平安、镰仓时代）的考古或传世材料。

2. **分类参照对象**：
   - 日用器物：如银碗托子、青瓷唾壶、注子注碗
   - 家具：如唐柜、椅子、脚床子
   - 装饰母题：如妇人启门、金银锭纹

3. **精确标注时间对应关系**，例如：
   - “朝鲜高丽时代（北宋—元）”
   - “日本贞观时代（晚唐）”
   - “日本弘安三年（元至元十七年）”

4. **明确使用目的**：
   - 验证中国器物对周边地区的输出影响
   - 辨识外来文化因素如何被本地接受并本土化
   - 构建以中国为中心的东亚共同物质文化圈

5. **确保材料来源权威可靠**，优先采用博物馆藏品写真帖、正仓院文书等正式出版物。

> 注意：仅在存在明确可比遗存且时间对应清晰时启用此技能。
