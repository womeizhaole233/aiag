# 宋辽建筑与墓葬中柿蒂纹识别与变体分析

## 触发条件
当构件上出现四瓣花形纹样，且满足以下前提时启动本技能：
- 存在彩绘或雕刻装饰；
- 纹样可明确辨识为四瓣对称花形。

> 注意：本技能不适用于非四瓣对称花形纹样。

## 执行步骤

1. **确认基本形态**：检查纹样是否为四瓣花形，通常呈对称分布。

2. **判断花瓣朝向**：
   - 若瓣尖朝内（如白沙宋墓、敦煌唐窟边饰、林东辽庆陵东陵），记录为“内向”；
   - 若瓣尖朝外（如《营造法式》图样及多数实例），记录为“外向”。

3. **定位应用构件**：根据纹样所在位置匹配典型用例：
   - **柱头**：常见于白沙墓、李昇墓、日本中尊寺金色堂（后者为“一整两破”形式）；
   - **小斗**：常绘四分之一柿蒂，与《营造法式》卷三十四五彩装净地锦小斗一致；
   - **大斗**：上部绘完整或四分之一柿蒂，歆部绘半柿蒂或四分之一柿蒂（见李昇墓）；
   - **棋**：白沙墓与义县奉国寺辽代彩画相似；
   - **散斗**：部分为“菱纹柿蒂”，不见于《营造法式》，但与辽庆陵东陵相似。

4. **识别特殊组合**：若为合斗，检查是否上部与歆部绘一完整柿蒂，四隅各绘四分之一柿蒂（见辽庆陵东陵）。

5. **推断时代与地域**：结合花瓣朝向、构件类型及组合方式，参考历史传播路径（晚唐盛行，传至辽地及日本藤原时代）进行判断。

6. **输出结论**：确认是否为柿蒂纹，并说明其方位特征、组合逻辑及可能的时代与文化区域（中原、辽、日本等）。

## 关键变量
- `petal_orientation`（花瓣朝向）：内向 / 外向
- `structural_element`（构件类型）：柱头、柱脚、小斗、大斗、棋、散斗等
- `cultural_region`（文化区域）：中原、辽、日本等
