# “宿白集”出版规范与文本处理原则

## 适用场景
当出版或再版类似《宿白集》的学术著作合集，且需在保留原貌的前提下提升文本质量与可读性时，应用本技能。

## 执行步骤

1. **底本选择**：以文物出版社原版为底本，维持内容基本不变。
2. **版式统一**：统一开本与版式设计，确保全书视觉一致性。
3. **图版更新**：替换或优化部分插图质量，提升图像清晰度与印刷效果。
4. **文字校订**：由北京大学考古文博学院师生修正初版排印错误。
5. **语言保留原则**：
   - 尽量保留初版语言词句原貌；
   - 对体例不一、讹脱倒衍等明显错误予以改正。
6. **引文校核规则**：
   - 若存在现行点校本，依点校本校核；
   - 若无点校本，依通行本校核。
7. **用字规范**：
   - 一般采用通行字；
   - 保留少数异体字，尤其涉及专业术语或原始文献时。
8. **补充与标注规范**：
   - 为阅读便利补入略去内容时，补文加（）；
   - 异文及作者说明性文字加（）；
   - 缺文或难辨文字以“口”表示；
   - 保留原始文献每行截止标记。

## 输出结果
生成符合学术规范、兼顾可读性与历史真实性的再版文本。
