# 第三号墓铺作构件尺寸详表

## 一、墓门铺作构件尺寸（单位：厘米）

### 斗类
- **枰斗**：上宽20，下宽15.5，耳高5.5，平高6，歙高4，总高15.5；斗歙无内龋
- **泥道棋上散斗**：上宽16，下宽12.2，耳高5，平高2，歙高3，总高10；斗歙内龋
- **慢棋上散斗**：上宽16，下宽12.2，耳高4.2，平高1.3，歙高4.7，总高10.2；斗歙内龋

### 棋类
- **泥道棋**：长52.3（棋端卷杀）
- **泥道慢棋**：长67（棋端卷杀）
- **昂**（柱头铺作）：长23（琴面）
- **耍头**：长22（蚂蚱头）
- **华棋**（补间铺作）：长15（棋端卷杀）

## 二、墓室转角铺作构件尺寸（单位：厘米）

### 斗类
- **栌斗**：上宽20，下宽16，耳高5，平高1，歙高4，皿板高5，总高15；斗歙内龢，皿板与栌斗底同宽
- **散斗**：上宽15.2，下宽13，耳高5，平高1，歙高4，总高10；斗歙内龢

### 棋类
- **泥道棋**：长74（棋端卷杀）
- **泥道慢棋**：长90（棋端卷杀）

## 三、历史注释
- (121) “不出跳”处理是砖建筑性能的适应性简化
- (123) 皿板见于北朝石刻，北宋仍存，但宋代皿板与牯斗底同宽，区别于北朝宽大形制
