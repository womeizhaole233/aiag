# 分析北宋小木作流行的社会经济动因

## 何时使用
- 研究北宋中后期建筑装饰风格（如佛道帐、藻井、壁藏等）时
- 解释商铺门前“闹斗八”“缚彩建欢门”等临时或永久性装饰的成因
- 分析墓葬中摹仿佛教/道教建筑的小木作元素与其墓主身份关联
- 遇到《东京梦华录》《宋会要辑稿》等文献记载的建筑装饰现象需社会经济解读

## 执行步骤
1. **确认适用对象**：仅限城市建筑、寺观、商铺、墓葬四类，排除战乱或经济衰退区域案例。
2. **识别两大核心动因**：
   - 高级统治阶层（皇室以下）对殿堂装饰繁缛的需求
   - 寺观广泛采用小木作构建佛道帐、经藏、壁藏等宗教设施
3. **重点考察商人阶层推动作用**：
   - 商人为彰显“屋宇雄壮”，在店铺大量使用繁缛小木作
   - 具体表现包括：
     - 门前设置“闹斗八”
     - “缚彩建欢门”（如酒楼装饰）
     - “架枋木花样结山棚”（食店装饰）
     - “两廊建小阁子”
     - “设飞桥栏槛，明暗相通”（如白矾楼）
4. **结合政策反证**：引用《宋会要辑稿》景祐三年诏书禁止非临街建筑使用“闹斗八”，佐证其在商铺中的普遍性。
5. **关联墓葬证据**：若墓主疑为地主兼商人，将其墓中小木作（如佛道帐、藻井）与同期城市商业繁荣背景进行交叉分析。

## 输出要求
将装饰风格、建筑类型与墓主/业主社会身份及北宋中后期商业经济背景建立逻辑关联，形成因果解释。
