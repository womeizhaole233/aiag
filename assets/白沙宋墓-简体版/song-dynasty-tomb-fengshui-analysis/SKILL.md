# 宋代墓葬风水分析

## 适用条件
- 分析对象为宋代墓葬
- 墓葬选址明显受到传统风水（堪舆）观念影响
- 可获取墓葬所在区域的地形、水文及方位信息

## 执行步骤

1. **确认地理位置特征**
   - 描述墓葬所处宏观地貌：是否位于山南水北、封闭谷地或盆地
   - 标识关键地形要素：后方是否有连绵山岗（如“走马岗”），前方是否有水域（如“饮马塘”）
   - 记录四象方位对应地貌：白虎（右）、青龙（左）、朱雀（前）、真武/玄武（后）

2. **对照《地理新书》标准**
   - 引用《地理新书》卷二《地形吉凶》：“后有走马岗，前有饮马塘，冈阜形势，小顿大起，延连百里不断者，为上吉。”
   - 检查是否满足“白虎缘山，青龙入泉，朱雀鼓翼，真武登天”的格局

3. **作出风水判定**
   - 若上述要素齐备且符合宋代堪舆逻辑，则判定为“上吉之地”
   - 否则说明缺失或冲突项，并指出不符合的具体标准

> 注意：本技能仅适用于受北宋官方风水体系影响的墓葬，不适用于民间异俗或非汉文化圈墓葬。
