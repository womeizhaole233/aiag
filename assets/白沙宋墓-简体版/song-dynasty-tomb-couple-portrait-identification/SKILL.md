# 宋代墓主夫妇像识别规范

## 适用条件
仅适用于满足以下全部前提的墓室壁画场景：
- 存在成对男女坐像
- 配有家具（朱色椅、横置桌、屏风）及侍从
- 布局对称，非单人像或非对称构图

## 执行步骤

1. **确认主人像特征**
   - **男性**：戴青幞头（前二脚结胡桃结），着圆领青袍，袖手坐朱色椅，背后设屏风（屏心有墨书四行，字迹草率）。
   - **女性**：梳高髻，戴黄色团冠，插簪饰，面颊施淡红、点朱唇，着红缘蓝衫，袖手坐朱色椅。

2. **核查家具配置**
   - 桌为朱色，横置（区别于砖砌顺放桌）。
   - 桌面由四条边梃框镶独板板心，边梃相交采用45度格角榫。
   - 桌上器物包括：白注子置于莲瓣注碗内；白碗衬红托子；黑盘盛朱果七枚；另二黑盘分盛果品四枚、六枚。

3. **验证侍从配置**
   - **女侍**：高髻戴朱缘黄团冠，着蓝缘朱衫，双手击拍板，面北侍立。
   - **男侍**：梳童髻系青带，着圆领淡黄衫，行叉手礼，面南侍立。

4. **综合判定依据**
   - 与已知墓例（如白沙宋墓第一号墓前室西壁）布置一致。
   - 位于墓室西南壁核心位置。
   - 具备完整生活场景要素（家具、饮食、乐舞、仪仗）。

5. **输出结果**
   - 若全部条件满足，提示：“该组人物为墓主人夫妇，可用于身份判定与社会地位分析”。

> **注意**：若缺少任一前提条件（如无侍从、单人像、非对称布局），不得应用本技能。
