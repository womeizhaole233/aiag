# 判断家具由席床向地面转移的证据

## 使用时机
当分析古代墓室（尤其是汉至南宋时期）的壁画、砖雕或实物遗存时，若同时满足以下条件，则启用本技能：
- 墓室中存在明确的床榻结构
- 存在可识别的独立家具图像或遗迹
- 墓室非无床榻类型（如部分简葬墓）

## 执行步骤
1. **确认家具原属席床**：检查以下家具是否出现在地面位置而非床上：
   - 衣架
   - 巾架
   - 镜台
   - 矮足柜
2. **识别新兴地面家具**：判断是否存在此前未见于席床之上、且功能适配站立或高坐姿态的新式家具，例如：
   - 曲足盆架
   - 高几
   - 高足柜
3. **关联生活习俗变迁**：结合垂足坐姿普及的历史背景，评估室内床外空间扩大是否催生了此类家具。
4. **引用文献佐证**：参考南宋《老学庵笔记》等史料中关于高镜台“禁中常用，外间不复用”的记载，强化家具下地趋势的判断。

## 输出结论
若发现上述地面独立家具，即可得出结论：“发现地面独立家具即表明已进入高坐时代”。
