# 宋代砖室壁画墓发掘流程

## 触发条件
发现带彩绘的宋代砖室墓，且满足以下前提：
- 墓室已部分暴露
- 具备测绘与文物保护条件

## 执行步骤

### 1. 初步发现与现场控制
- 立即停止周边施工
- 封回已揭开封顶砖并覆土保护
- 报告文物主管部门（如省级文物保管委员会）

### 2. 正式发掘准备
- 沿墓顶外围谨慎取土，探查是否存在相邻结构（如前后室）
- 记录墓葬与生土之间的空隙（通常为9–10厘米）
- 测绘墓道位置与结构

### 3. 墓道与封门清理
- 清理墓道，记录阶梯数量与起始位置
- 注意墓道两侧可能出土的遗物（如白瓷碗碎片）
- 按层拆除封门砖：
  - 外层：横砖与菱角牙子混砌
  - 中层：全横砖
  - 内层：全卧丁砖

### 4. 墓室内部清理
- 清除淤土，注意浮土来源（如人为扰动）
- 重点检查过道壁面，寻找纪年题记（如“元符二年”）
- 系统收集出土物：陶瓦残片、铁钉、碎瓷、地券、铜钱（如“绍圣元宝”）
- 对结构不稳定部位（如崩毁壁面）立即加装临时支撑（立柱+过梁）

### 5. 记录与收尾
- 完成照相、测绘与壁画临摹
- 安装保护性木门
- 提交完整发掘报告

> **注意**：本流程仅适用于已部分暴露、结构尚存的宋代砖室壁画墓。若墓葬完全坍塌或未暴露，应采用其他勘探或评估方法。
