# 《白沙宋墓》技能集

本技能集基于考古学经典著作《白沙宋墓》（书号：E50696Bc-4159-46D0-8037-6E43C84Bc3A6），系统整合了宋代墓葬研究中的53项核心技能，涵盖田野发掘、建筑结构分析、壁画与随葬品解读、风水制度、区域经济、工艺技术、文献处理及跨文化比较等多个维度。这些技能不仅适用于北宋晚期至元代墓葬的专项研究，也为理解宋代社会生活、丧葬观念与物质文化提供了方法论支撑。
