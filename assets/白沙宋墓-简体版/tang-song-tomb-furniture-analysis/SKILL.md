# 唐宋墓葬家具陈设分析技能

## 适用条件
- 墓室壁面存在可辨识的家具图像或砌筑结构
- 家具类型包括椅、桌、机（矮凳）、脚床子等高足家具
- 不适用于仅置棺椁而无家具陈设的早期墓葬

## 执行步骤
1. **识别家具类型与位置**：记录墓室各壁面所绘或砌筑的家具种类及数量（如前室西壁二椅一桌、后室东壁一椅等）。
2. **判断年代特征**：若出现高足家具（尤其是垂足坐用的椅、桌），则初步判定该墓葬年代不早于五代宋初，且极可能为北宋中叶（约11世纪后期）以后。
3. **结合旁证验证**：参考同期考古发现（如安阳王用墓壁画、钜鹿城出土桌椅实物）及传世绘画（如《清明上河图》《韩熙载夜宴图》）中高足家具的普及程度，强化年代与社会生活判断。
4. **推断起居方式变革**：确认该墓葬反映的是从汉唐跪坐/盘坐于席榻，向垂足坐椅、面前设高桌的起居方式转变，标志室内布置与日常生活发生重大跃进。

## 输出结果
生成关于墓葬年代范围及对应社会起居方式变革的判断结论，格式为：“通过家具类型判断，该墓葬应属北宋中叶以后，反映了从席地而坐向垂足高坐的起居方式变革。”
