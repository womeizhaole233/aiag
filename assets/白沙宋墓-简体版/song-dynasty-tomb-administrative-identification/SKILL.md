# 宋代墓葬历史行政区划判定

## 适用条件
- 出土文物中包含明确记载地名的墓志铭或买地券
- 墓葬年代初步判定为宋代（960–1279年）
- 需要精确还原墓葬在宋代所属的府、县、乡级行政区

## 执行步骤
1. **提取关键出土文书信息**
   - 记录墓志或地券中出现的完整地名表述（如“西京登封县天中乡嵋中村”）
   - 标注文书纪年（如政和四年、宣和六年）

2. **查证该地名的历史沿革序列**
   - 按时间顺序梳理从先秦至明代该地的行政归属变化
   - 重点关注北宋时期（960–1127）的区划设置

3. **匹配文书年代与历史区划**
   - 将出土文书纪年对应到历史沿革中的具体时段
   - 确定该时段内该地所属的府、县、乡三级行政单位

4. **输出结论**
   - 明确表述墓葬在宋代的完整行政归属
   - 注明现代地理位置以供对照

> 注意：仅适用于有明确出土文字证据的遗址；若无直接地名记载，不得推测使用。
