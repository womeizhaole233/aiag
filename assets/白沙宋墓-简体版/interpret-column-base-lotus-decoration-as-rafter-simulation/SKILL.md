# 解读柱脚莲花纹作为柱檩彩画的象征性模仿

## 何时使用
- 柱脚单独绘制莲花纹，柱头和柱身中部无莲花装饰
- 研究对象为墓室倚柱或传统建筑柱子
- 存在《营造法式》所载柱檩彩画制度背景
- 需解释墓葬装饰与地上建筑构件之间的象征关联

## 执行步骤
1. **确认莲花纹位置**：核实莲花仅出现在柱脚，排除柱头或柱身中部的莲花纹。
2. **核查建筑传统**：确认该时期或地域存在柱檩结构及檩条莲花彩画传统（如《营造法式》“五彩遍装”或“碾玉装”制度）。
3. **比对实物证据**：参考已知束莲石檩等实物（如云冈石窟辽金遗物），验证莲花装饰在真实建筑构件中的应用。
4. **推断象征逻辑**：基于“事死如事生”观念，判断柱脚莲花纹是对无法在墓中设置真实檩条的替代性摹拟。
5. **输出解释结论**：明确指出柱脚莲花纹是对地上建筑柱顶檩条莲花装饰的象征性再现。

> 注意：若柱头同时有莲花纹，则不适用本技能，应视为独立装饰体系。
