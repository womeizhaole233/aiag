# 宋辽墓顶明镜象征识别

## 何时使用
- 墓顶中央绘有孤立白地圆形图像
- 图像周围无藻井或其他装饰
- 墓葬年代属于宋辽时期（含可能承袭唐代传统的案例）
- 不适用于具有复杂藻井结构或非圆形顶心装饰的墓葬

## 执行步骤
1. **确认图像特征**：检查墓室顶心是否为墨绘白地圆形物，且周围无其他装饰。
2. **比对文献记载**：参照《营造法式》卷八《小木作制度·小斗八藻井》中“顶心之下……内安明镜”的描述。
3. **核查实物佐证**：确认是否存在宋代或辽代墓葬中悬挂铜镜或绘制明镜的实例。
4. **判断象征性质**：若为绘制而非实物悬挂，则判定为象征性明镜表达。
5. **推断功能与意义**：结合时代背景，推测其可能具有驱邪、照冥、象征天穹或灵魂归所等功能。
6. **输出结论**：提示该图像为明镜象征，反映宋辽时期墓葬宇宙观与丧葬信仰。

> 注意：本技能仅适用于符合触发条件的孤立圆形顶心图像，不适用于装饰繁复或结构不同的墓顶。
