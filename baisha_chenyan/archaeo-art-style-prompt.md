# 考古解谜游戏统一画风生图提示词

> 基于《白沙宋墓》矿物质感风格提炼。适用于人物、场景、背景、UI界面、道具、地图等各类游戏素材的风格统一生成。素材为独立插画，非墙面壁画。

---

## 一、核心风格锁（所有题材共用，禁止修改）

无论生成何种题材，以下风格要素必须保留，以确保全游戏视觉统一：

| 风格要素 | 描述 | 提示词关键词 |
|----------|------|-------------|
| **色调** | Sepia 基调，矿物颜料褪色感 | `sepia-toned`, `faded mineral pigment`, `restrained palette` |
| **线稿** | 精确墨线/深褐色线，清晰锐利 | `precise ink lines`, `dark brown outlines`, `sharp linework` |
| **上色** | 极淡透明色块，不覆盖线稿 | `very subtle color washes`, `transparent mineral tint` |
| **质感** | 矿物质感，如褪色古画/文献插图 | `mineral pigment texture`, `aged paper quality` |
| **背景** | 干净简洁，无墙面纹理 | `clean background`, `off-white`, `no wall texture` |
| **光源** | 左上方柔光，微弱体积感 | `soft top-left lighting`, `gentle volumetric light` |
| **氛围** | 古朴、静谧、学术感 | `scholarly illustration`, `museum documentation quality` |
| **排除** | 无文字/标注/水印 | `no text`, `no labels`, `no watermarks` |

## 四、风格一致性保障清单

批量生成不同题材素材时，使用以下固定参数锁定风格：

| 保障项 | 操作建议 |
|--------|----------|
| **固定 Seed 值** | 同一批素材使用相同或相近 Seed |
| **参考图锁定** | 以第一张满意的图作为后续 Reference Image |
| **色板固定** | 矿物色不超过：淡赭石、淡石青、淡石绿、淡青灰、淡暖褐、淡墨灰 |
| **线稿统一** | 全部使用深褐色/墨色线，避免彩色线条 |
| **背景统一** | 全部使用干净白灰色渐变背景，避免墙面纹理 |
| **后期统一** | 生图完成后统一叠加 Sepia 滤镜 + 细微噪点，确保风格一致 |

## 六、快速工作流

1. **确定题材**：选择对应模板（A-F）
2. **替换变量**：填写 `{大括号}` 中的内容
3. **套用风格锁**：确认核心风格要素完整保留
4. **生成测试图**：先跑1张验证风格
5. **锁定参数**：记录 Seed / 参考图 / 色板
6. **批量生成**：统一参数输出全部素材
7. **后期统一**：叠加 Sepia 滤镜 + 细微噪点，确保风格一致

---

## 七、注意事项

- **AI 会篡改结构**：生图仅作氛围参考，线稿类素材建议用原始线稿叠加处理
- **颜色过浓**：若 AI 上色太深，加入 `very subtle`, `transparent washes`, `faded`, `ghostly remnants` 压低
- **文字残留**：若生成图中有奇怪符号，属于 AI 幻觉，直接去除
- **UI 文字**：UI 模板已排除文字内容，实际游戏中的文字由程序层叠加，保持素材纯净
- **人物变形**：人物生成易出现面部/手部问题，建议配合 ControlNet 线稿约束，或后期用 PS 修正
- **背景干净**：所有素材均为独立插画，背景应保持简洁，方便游戏中叠加使用
