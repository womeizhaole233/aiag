# 古墓探秘游戏项目进度文档

## 项目概述
本项目是一个以古墓探险为主题的HTML5网页游戏合集，包含消消乐挖土游戏和文物擦灰游戏两个核心玩法模块。

## 一、已完成工作

### 1. 消消乐挖土游戏（match-3）

#### 核心玩法
- [x] 11×12网格布局（竖11层 × 横12层）
- [x] 6种方块类型：泥土、碎石、残片、灰尘、砖块、垃圾
- [x] 3个及以上相同方块相邻即可消除
- [x] 玩家通过点击交换相邻方块来触发消除
- [x] 累积消除100块方块获得胜利
- [x] 胜利后展示获得文物并自动重置游戏

#### 重力下落系统
- [x] 被消除方块上方的方块独立下落
- [x] 新方块在画布上方生成并下落
- [x] 每个方块下落距离 = 该方块下方被消除的方块数量
- [x] 下落总时长固定为0.5秒
- [x] 使用easeOutCubic缓动，动画流畅自然
- [x] 连锁消除支持（下落后自动检测新匹配并继续消除）

#### 动画系统
- [x] 匹配方块高亮闪烁（0.5秒）
- [x] 下落动画阶段
- [x] 非下落方块保持可见
- [x] 动画期间禁止玩家操作
- [x] 支持中途重置取消动画

#### 进度系统
- [x] 右侧垂直进度条
- [x] 每次消除累积计数
- [x] 进度条动画过渡（0.8秒）
- [x] 累积消除100块时触发胜利

#### 美术资源
- [x] 从PDF提取6种方块图片素材
- [x] 图片资源统一存放在 `picture/` 文件夹
- [x] 古墓主题UI风格（石椁、火把、符文等装饰）

#### 其他功能
- [x] 重置游戏按钮
- [x] 退出游戏按钮
- [x] Toast提示消息
- [x] 游戏状态管理（处理中、进行中、胜利等状态）

## 三、技术栈
- HTML5 + CSS3 + JavaScript（ES6+）
- Canvas 2D 渲染
- 异步动画（Promise + requestAnimationFrame）
- 本地图片资源加载
