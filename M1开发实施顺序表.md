# M1 开发实施顺序表

> 本文用于安排《白沙宋墓 M1》游戏后续开发优先级。原则：先解决空间迷失，再补场景结构，再补线索推理，最后加入小游戏和 NPC 叙事。

## 推荐总顺序

1. 定位导航图。
2. 前室、后室入口总视角。
3. 后室日常器物线。
4. 前室礼仪细节补强。
5. 结论卡与线索墙。
6. NPC 基础剧情。
7. 小游戏。
8. 素材清单长期维护。

## 第一阶段：解决空间迷失

目标：让玩家知道自己在墓室哪里。

| 内容 | 设计 |
|---|---|
| 必须做 | 增加整体定位导航图 |
| 加入图片 | `一号墓平剖面图.png`、后续可补 `第一墓室剖面结构图.png` |
| 场景影响 | 墓门、甬道、前室、过道、后室都能查看定位图 |
| 线索层级 | E 环境线索、P0 空间定位线索 |
| 小游戏 | 第一版不做；后续可加入“平剖面定位” |
| NPC | 考古领队说明“先确认空间，再谈图像”；考古技工提示“入口、墙面、顶部要分开记” |
| 完成标准 | 玩家随时知道当前位置、朝向、前后关系；进入前室/后室不会迷失 |

## 第二阶段：补充大场景入口总视角

目标：把场景从“图片切换”变成“进入一个空间”。

| 内容 | 设计 |
|---|---|
| 必须做 | 前室入口总视角、后室入口总视角 |
| 加入图片 | 前室可用入口相关图或平剖面辅助；后室可用 `第一号墓过道北壁下部一一后室入口.png`、`第一号墓后室南壁一一后室入口背面.png` |
| 场景影响 | 前室不再直接进入西壁；后室不再直接进入北壁 |
| 线索层级 | E 空间线索、P1 场景组织线索 |
| 小游戏 | 暂不做复杂小游戏，只做“选择观察方向”：左看、右看、前看、抬头 |
| NPC | 考古队员提示“先拍入口总照，再分墙面记录” |
| 完成标准 | 每个章节都像独立关卡空间；玩家知道自己从哪里进、往哪里看、如何返回 |

## 第三阶段：补后室日常器物线

目标：把后室从“北壁 + 出土物”扩展成完整后室章节。

| 内容 | 设计 |
|---|---|
| 必须做 | 后室南壁、西南壁、东北/西北壁、顶部 |
| 加入图片 | `第一号墓后室南壁壁画中的高几.png`、`第一号墓后室西南壁壁画中的镜台.png`、`插图三五 第一号墓后室西南壁壁画中的杌.png`、`插图三六 第一号墓后室西南壁壁画中的曲足盆架.png`、`第一号墓后室东北壁壁画中的灯菜.png`、`第一号墓后室西北壁壁画中的细腰剪刀、熨斗.png`、`第一号墓后室顶.png` |
| 场景影响 | 后室新增多视角：南壁、西南壁、东北壁、西北壁、顶部 |
| 线索层级 | P0：镜台/盆架/杌组合；P1：高几、灯菜、剪刀熨斗；P2：顶部铺作、墙面残损 |
| 小游戏 | 日常器物归类：把器物归入“梳洗、照明、熨烫、陈设” |
| NPC | 考古队员协助编号；教授在收束时指出“后室图像出现日常化倾向” |
| 完成标准 | 玩家获得“后室日常化叙事”结论卡 |

## 第四阶段：补前室器物与礼仪线

目标：让前室不只是五个视角，而是能推理出“礼仪空间”。

| 内容 | 设计 |
|---|---|
| 必须做 | 东壁女乐细节、西壁器物细节、南壁壁函细节 |
| 加入图片 | 莲花冠、排箫、尖鞋、唾壶、金银铤、碗托、脚床子、椅子、骨朵 |
| 场景影响 | 前室现有五视角保持，但增加近景证据图 |
| 线索层级 | P0：女乐、注子/高瓶、壁函、顶部铺作；P1：莲花冠、排箫、唾壶、金银铤；H：尖鞋不单独说明身份 |
| 小游戏 | 礼仪归类：把人物、器物、建筑构件归入“奏乐、陈设、通行、结构” |
| NPC | 教授提示“不要用单一服饰判断身份”；考古队员提示“同组图像要合看” |
| 完成标准 | 玩家获得“前室礼仪空间与年代错层”结论卡 |

## 第五阶段：建立结论卡与线索墙

目标：让玩家知道所有线索最终指向什么。

| 内容 | 设计 |
|---|---|
| 必须做 | 结论卡系统、线索墙系统 |
| 加入图片 | 可用平剖面图作为背景，不一定加新图 |
| 场景影响 | 每章完成后生成一张结论卡 |
| 线索层级 | 把 P0/P1/H/E 线索汇总为 R01-R09 推理链 |
| 小游戏 | 线索连线：把线索卡拖到对应结论上 |
| NPC | 教授负责总结；考古队员负责提醒缺证据 |
| 完成标准 | 玩家能回答“这些线索最终说明了什么” |

## 第六阶段：加入剧情与 NPC 对话

目标：让探索有节奏，但不破坏学术感。

| 内容 | 设计 |
|---|---|
| 必须做 | 开场对话、章节入口对话、章节完成对话 |
| 加入图片 | 环境地图、墓外图、平剖面图 |
| 场景影响 | 墓外阶段正式成立 |
| 线索层级 | E 环境线索、剧情触发线索 |
| 小游戏 | 墓外定位：根据村民提示找到墓群位置 |
| NPC | 附近村民、考古技工、考古队员、考古领队 |
| 完成标准 | 玩家明确身份是考古实习生，并理解自己为什么进入 M1 |

## 第七阶段：增加中后期推理小游戏

目标：提高玩法，但不让小游戏喧宾夺主。

| 内容 | 设计 |
|---|---|
| 必须做 | 题记辨读、遗物定位、器物归类、线索连线 |
| 加入图片 | 题记近景、出土物分布图、器物特写、平剖面图 |
| 场景影响 | 过道、后室、终章增强 |
| 线索层级 | P0/P1 线索通过小游戏确认；H 线索通过小游戏排除 |
| NPC | 教授给判断标准；技工提示观察方法；队员提示记录流程 |
| 完成标准 | 玩家不是只点击收集，而是参与判断 |

## 第八阶段：补齐未使用图片与备用材料

目标：让素材库有秩序，不再边做边乱。

| 内容 | 设计 |
|---|---|
| 必须做 | 资产使用清单、备用图清单、未使用图清单 |
| 加入图片 | `11`、`12`、`13`、`15`、`17` 目录中未进入 assets 的图片 |
| 场景影响 | 后室、前室、环境章节持续补强 |
| 线索层级 | P1/P2/H 为主 |
| 小游戏 | 暂不新增，根据需要补 |
| NPC | 无需新增 |
| 完成标准 | 每张图片都有状态：已用 / 待用 / 备用 / 不适合当前版本 |
