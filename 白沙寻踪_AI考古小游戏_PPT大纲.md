> 考古题材沉浸式点击解谜游戏。

---

## P1 封面页

**主标题：** 白沙寻踪——AI考古小游戏

**副标题：** 考古题材沉浸式点击解谜游戏

**一句话定位：** 以白沙宋墓第一号墓（M1）真实考古材料为核心，把学术文本转化为可探索、可记录、可推理、可叙事的互动体验。

**视觉建议：** 墓门 / 妇人启门 / “元符二年赵大翁布”题记三者拼贴，暗角 + 手绘考古笔记质感。

## P2 痛点与机会

**标题：** 真实考古材料很好，但大众“进不去、读不懂、连不上”

### 四大痛点

1. **文博体验“看而不得入”：** 博物馆展陈仍以静态观赏为主，观众与文物隔着玻璃、展板和专业术语。
2. **考古教育缺乏沉浸式工具：** 高校教学依赖图文讲授和实地参观，难以完整体验考古推理过程。
3. **解谜游戏内容同质化严重：** 中式恐怖解谜依赖民俗恐怖和虚构灵异，缺乏学术底蕴，口碑下滑。
4. **文化遗产数字化停留在“资料库”层面：** 3D扫描、虚拟展厅有信息无行动、有展示无推理、有数据无叙事。

### 市场机会

- 中国游戏市场2024年实际销售收入 **3,257.83亿元**，用户规模 **6.74亿**。
- 中国悬疑解谜游戏市场2025年预计突破 **280亿元**，2030年预计突破 **200亿元**，年复合增长率 **12%**。
- 全球教育游戏市场2024年达 **122.4亿美元**，预计2032年达 **215.8亿美元**，CAGR **38.67%**。
- 中国数字教育市场2022年 **4,538亿元**，年增长率 **12.3%**。
- 文博消费从“打卡”转向“文化认领”，年轻用户渴望从旁观者变为参与者。

## P3 解决方案：产品本身是什么

**标题：** 不是资料库，也不是纯虚构机关解谜——而是一部“可玩的考古发掘笔记”

### 产品形态

- Web端 + 移动端H5 + 微信小程序。
- 以**白沙宋墓M1真实考古图片、线索表、映射表**为舞台。
- 玩家扮演**考古实习生**，在考古队工作流程中逐步学习观察、记录、复查和判断。

### 产品定位

| 维度 | 内容 |
|---|---|
| 产品类型 | 考古题材沉浸式点击解谜游戏 |
| 核心素材 | 白沙宋墓M1真实考古图片、线索表、映射表 |
| 玩家身份 | 考古实习生 |
| 核心体验 | 以墓葬图像为全屏底图，点击热区获得观察信息，记录线索，形成推理判断 |
| 风格关键词 | 学术、科学、克制、悬疑、沉浸 |
| 差异化定位 | 不做只堆图片的资料库，不做脱离考古材料的纯虚构机关解谜 |

### 核心玩法循环

**观察 → 记录 → 复查 → 推理 → 推进**

进入场景 → 观察全屏底图 → 点击隐藏线索热区 → 获得事实性观察文案 → 线索自动进入记录夹 → 玩家在记录夹中复查、降级、汇总 → 形成章节结论 → 解锁下一空间。

### 章节结构（7章）

| 章节 | 主要内容 | 核心推理目标 |
|---|---|---|
| **墓外/环境** | 墓群位置、环境图、村民对话 | 建立调查背景与空间入口 |
| **墓门** | 门额、门额背面彩画、封门砖、门前地面 | 判断墓门结构与封闭痕迹 |
| **甬道** | 顶部、东壁、西壁三视角，叠胜彩画 | 理解通道空间与图像导向 |
| **前室** | 入口总览+五视角，女乐、器物、壁函 | 理解礼仪空间与年代错层 |
| **过道** | 纪年题记、破子棂窗、顶部宝盖 | 识别时间信息与空间转换 |
| **后室** | 多墙面+顶部，妇人启门、地券、人骨铁钉 | 综合图像、遗存、文书和日常器物线 |
| **终章** | 汇总各章节结论卡、关系链、线索墙 | M1多层材料共同指向的综合理解 |

### 两条设计红线

1. 不做只堆图片和说明的**静态资料库**。
2. 不做脱离考古材料的**纯虚构机关解谜**。

## P4 产品核心功能

**标题：** 一套证据系统 + 一支考古队 + 三项考古动作小游戏

### 一、线索系统（七级分级）

线索严格以M1的图片、线索表和映射表为准，分为七个层级：

| 层级 | 含义 | 示例 | 流程作用 |
|---|---|---|---|
| **P0** | 核心推进线索 | 门额背面彩画、纪年题记、妇人启门、地券 | 必需，章节推进和核心结论 |
| **P1** | 关键解释线索 | 破子棂窗、顶部宝盖、前室铺作 | 重要，通常不单独卡关 |
| **P2** | 沉浸观察线索 | 墙面残损、色层、砖缝、局部器物 | 补充，增强现场感 |
| **H** | 误导/待验证线索 | 尖鞋、手部断口、流苏方向 | 待验证，需降级或排除 |
| **E** | 环境线索 | 平剖面图、入口、墓道环境 | 定位，不进入记录夹 |
| **R** | 推理关系链 | 章节结论、跨章节关系 | 结论卡和终章汇总 |
| **F** | 剧情虚构线索 | 银扣、暗格、铜筒、绢帛 | 仅服务剧情，不作为考古证据 |

### 二、记录夹：从点击到推理的桥梁

记录夹不仅自动收录观察记录，还支持：

- **复查：** 把多条相关线索重新放在一起看。
- **降级：** 把误导性线索处理为服饰细节、残损等。
- **汇总：** 形成章节判断或结论卡。

玩家不是被动接受信息，而是主动管理自己的证据体系，最终感受到“我通过证据理解了这座墓”。

### 三、考古队群像NPC

NPC互动采用“短对话+选择项+任务反馈”模式，分为固定剧情和AI自由对话两类。

| NPC | 身份 | 主要作用 | 语言风格 |
|---|---|---|---|
| **附近村民** | 地方居民 | 提供地方记忆、发现经过、传闻氛围 | 口语、简短、带不确定性 |
| **考古技工** | 现场技师 | 提醒封门、砖缝、构件、施工痕迹 | 经验型、直接、重现场 |
| **考古队员** | 同事 | 提醒补拍、对照、归档 | 同伴协作，不直接给答案 |
| **考古领队/教授** | 导师 | 引导玩家组合线索成判断；AI自由对话NPC | 学术、克制、边界清楚 |

AI自由对话NPC（考古领队/教授）遵守严格约束：不能凭空生成考古事实、不能提前剧透后续章节、不能替玩家直接完成推理，默认不写入记录夹。

### 四、考古动作小游戏

小游戏严格绑定考古动作，产出直接进入记录夹：

1. **挖土消消乐：** 墓外/墓门前，消除土块、碎砖，逐步露出目标区域。
2. **题记辨读：** 过道阶段，根据题记近景完成文字辨读或选择，确认纪年题记。
3. **后室遗物定位：** 后室阶段，在分布图中定位地券、人骨、铁钉等遗存。

## P5 技术壁垒与AI应用

**标题：** 人文为核，AI为翼，学术边界是护城河

### 1. 真实考古素材壁垒

- M1图片、线索表、映射表、考古报告全部可追溯。
- 与虚构民俗解谜形成内容天花板差异。

### 2. “证据→线索→推理”转译机制

- 把学术文本转化为玩家可操作的线索、选项、小游戏和NPC对话。
- P0-P2/H/E/R/F分级体系把“学术边界意识”变成核心玩法。

### 3. AI受控内容转译（不是编故事）

- **NPC自由对话系统：** LLM+RAG驱动，回答严格基于本地知识库。
- **AI辅助线索文案生成：** 将学术原文改写为2-3句弹窗文案，经人工审核。
- **AI辅助内容生产：** 生成场景模块交付表、NPC剧情节点草稿、多语言本地化。
- **AI数据分析与用户洞察：** 分析点击热区分布、停留时长、卡点位置，优化关卡设计。

系统提示词约束：不能凭空生成事实、不能剧透、不能替玩家推理。

### 4. 可复用框架

- 场景数据与游戏逻辑解耦，可快速扩展M2/M3及其他遗址。
- 远期沉淀为“AI文博互动内容生产工具”。

## P6 应用场景演示

**标题：** 90分钟里，玩家会经历什么？

### 第一章：墓外/环境

- 玩家以考古实习生身份抵达白沙，与附近村民交谈，了解墓群发现经过和地方传闻。
- 考古技工凭经验判断“下面三尺是墓道填土”。
- 小游戏**挖土消消乐**：清理土块、碎石、残片，露出墓道和封门砖。

### 第二章：墓门

- 观察门额、门额背面彩画、封门砖、门前地面。
- 三层封门砖：外层横砖+菱角牙子，中层横砖，内层卧丁砖。
- 玩家判断拆除顺序，发现内层卧丁砖裂缝。
- 选择：继续拆 / 架支撑 / 叫考古队员。正确选择：架撑。
- 拓下“永安”二字，领队问含义——玩家选择“目前无法判断”才正确。
- 门额背面彩画揭示：**“正面看完，必须想背面。”**

### 第三章：甬道

- 版门赭色、门钉七排五钉、断砌造。
- 队员观察顶部叠胜彩画，技工判断叠涩结构。
- 领队引导“三面合读”：顶部给方向，侧壁给内容。

### 第四章：前室

- 穹窿顶藻井、东壁女乐、西壁墓主夫妇开芳宴、南壁侍卫。
- 理解礼仪空间与年代错层。

### 第五章：过道

- 小游戏**题记辨读**：从“元符二年赵大翁布”中确认1099年时间锚点。
- 关键教学：题记是锚点，不是答案。
- 识别破子棂窗、顶部宝盖等时间信息与空间转换线索。

### 第六章：后室

- 六角形后室，北壁假门，妇人启门。
- 系统提示：**当前证据支持“图像中的门”，不支持“实际可进入的门”。**
- 小游戏**后室遗物定位**：在分布图中定位人骨、十九枚铁钉、地券、镇墓钱。
- 队员判断：两具人骨头骨并列、其余骨骼混堆，倾向迁葬——但标为“推测，待核”。
- 发现暗格铜筒、素帛反切、绢帛自述，全部标为**F类剧情线索**，只能进入“可推测”或“仍存疑”。

### 第七章：终章

- 汇总各章节结论卡、关系链、线索墙。
- 完成线索墙连线，形成最终判断。
- 领队收尾：**“剧情线可以被情感回收，但不能反向改写P0结论。”**

## P7 商业模式

**标题：** B2C + B2B + B2G + IP衍生，先内容闭环，再商业变现

### 收入结构

| 收入来源 | 模式说明 | 目标客户 | 预期占比 |
|---|---|---|---|
| **B2C游戏发行** | 免费下载+章节解锁内购+增强包DLC+装饰性内容 | 解谜游戏玩家、文博兴趣者 | 50-60% |
| **B2B文博授权** | 博物馆定制线下互动体验版本 | 博物馆、文博机构 | 20-25% |
| **B2G教育采购** | 高校考古课程互动教材版本 | 高校考古专业、文博培训 | 10-15% |
| **IP衍生文创** | 周边、数字藏品、IP授权 | 文博消费群体 | 5-10% |

### B2C定价策略

- **基础版免费：** 墓门章节免费体验，建立用户认知和信任。
- **章节解锁：** 甬道-前室-过道-后室逐章解锁，单章定价 **6-12元**，全章打包 **38-58元**。
- **增强包DLC：** 终章深度解析、专家讲解音频、幕后考古纪录片，定价 **18-28元**。
- **装饰性内容：** 记录夹皮肤、特殊线索标记样式，定价 **1-6元**。

目标ARPPU **80-150元**。

### B2B文博授权

- **标准授权：** 博物馆购买产品授权，在展厅设互动体验区，按年收取授权费 **5-15万元/年**。
- **定制开发：** 基于博物馆特定藏品/遗址定制游戏内容，项目制收费 **20-50万元/项目**。
- **联合运营：** 与博物馆联合运营线上版本，流量导入线下展览，收入分成。

### B2G教育采购

- **课程版授权：** 面向高校考古专业，提供教学版（含教师后台、学生进度追踪），年费 **1-5万元/校**。
- **教育平台合作：** 接入智慧教育平台，按使用量计费。
- **研学项目：** 面向K12研学市场，开发简化版考古体验课程。

**当前重心：** 完成MVP内容闭环与学术背书，优先进入B端与教育场景。

- **对故纸修复师等线下体验：** 以数字产品形态突破物理限制，同时可反向输出给博物馆作为互动展陈方案。

## P12 风险分析（补充）

**标题：** 主要风险与应对

| 风险类型 | 风险描述 | 应对措施 |
|---|---|---|
| **学术准确性风险** | 游戏内容可能偏离考古事实，引发学术争议 | 建立学术顾问审校机制；线索系统严格区分考古证据(P0-P2)与剧情虚构(F)；所有文案需经专业审核 |
| **AI生成风险** | NPC自由对话可能生成不准确的考古信息 | RAG架构限制知识范围；禁止透露表和回复规则约束输出；默认不写入记录夹；标注“NPC提示”而非“考古证据” |
| **市场认知风险** | “考古”标签可能吓退普通玩家 | 营销重点放在“悬疑解谜”而非“考古教育”；墓门章节免费体验降低门槛 |
| **版权与合规风险** | 墓葬图片、考古报告的版权归属复杂 | 提前与考古研究所/文物出版社确认素材版权；使用公开发表资料时注明出处；原创文案保留著作权 |
| **开发延期风险** | 考古内容整理工作量超出预期 | MVP优先跑通单线流程；五阶段递进开发，每阶段独立可交付；AI辅助内容生产加速 |
| **竞品风险** | 大厂可能进入考古游戏赛道 | 先发优势+学术深度壁垒；与考古机构建立合作关系；快速迭代扩展IP矩阵 |
