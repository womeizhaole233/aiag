# M1 游戏整体结构与线索推理总表

> 本文用于约束《白沙宋墓 M1》点击式沉浸体验的整体结构、线索分层、推理目标、NPC 叙事与素材使用方向。后续新增场景、热区、小游戏和剧情时，应优先参照本文。

## 一、整体定位导航方案

定位导航不做常驻 UI，默认隐藏。玩家需要时点击“查看位置”热区弹出。

| 层级 | 建议用图 | 用途 |
|---|---|---|
| 全局定位图 | `M1/00_墓葬全景与结构图/一号墓平剖面图.png` | 标记玩家在墓门、甬道、前室、过道、后室中的位置 |
| 后室局部定位图 | `M1/00_墓葬全景与结构图/第一号墓后室平、剖面.png` | 后室多视角展开后，标记北壁、南壁、砖床、入口等关系 |
| 结构说明图 | `第一墓室剖面结构图.png`、`第一墓室墓顶情况.png` | 用于顶部结构说明或结构类小游戏，不作为日常导航主图 |

定位图需要显示：

- 当前所在区域。
- 当前朝向或观察对象。
- 已进入、已观察、已完成区域。
- 当前可继续探索的大方向。

## 二、章节结构总表

| 章节 | 当前状态 | 建议结构 | 核心推理目标 | 最终结论 |
|---|---|---|---|---|
| 墓外 / 环境 | 尚未正式制作 | 环境地图、附近图、村民 NPC | 白沙墓群位置、墓道朝向、进入前情 | 玩家确认这是一次考古调查，不是单纯探险 |
| 墓门 | 已制作 | 墓门正面、门额背面、封门砖 | 墓门结构与封闭痕迹 | 墓门既是入口，也是第一道证据界面 |
| 甬道 | 已制作 | 甬道东壁、西壁、顶部 | 通道空间与顶部彩画关系 | 甬道承担过渡与引导 |
| 前室 | 已较完整 | 建议补“入口看前室”总视角，再进入五视角 | 礼仪空间与年代错层 | 前室承担展示、礼仪、通行与结构转折功能 |
| 过道 | 已制作 | 东壁、题记、破子棂窗、宝盖 | 年代题记与空间转换 | 过道是时间信息和空间转换的交汇点 |
| 后室第一版 | 已制作 | 北壁、妇人启门、地券、人骨铁钉、分布图 | 后室身份、遗存与文书关系 | 后室集中墓主遗存、文书与图像核心 |
| 后室第二版 | 待制作 | 南壁、东西壁、西南壁、东北/西北壁、顶部 | 日常器物线与后室叙事 | 后室不只是安置遗体，也嵌入日常生活图像 |
| 终章汇总 | 待制作 | 线索墙、研究日志 | 多线索归并 | M1 的意义来自空间、图像、文书、遗物共同指向 |

## 三、线索分层

| 层级 | 类型 | 作用 | 示例 |
|---|---|---|---|
| P0 | 核心推进线索 | 推进章节或解锁结论 | 门额背面彩画、纪年题记、妇人启门、地券、人骨铁钉 |
| P1 | 关键解释线索 | 强化推理 | 破子棂窗、顶部宝盖、前室铺作、后室南壁高几 |
| P2 | 沉浸观察线索 | 增加现场感，不影响推进 | 墙面残损、边框、色层、砖缝、局部器物 |
| H | 误导线索 | 需要被排除或降级 | 流苏方向、灯菜位置偏移、单一尖鞋身份判断 |
| E | 环境线索 | 建立空间感 | 环境地图、墓道、入口、平剖面、顶面结构 |

## 四、章节推理目标

| 章节 | 主要线索 | 推理链 | 玩家获得的结论卡 |
|---|---|---|---|
| 环境 | 地形图、附近图、平剖面 | 墓葬位置 -> 墓道朝向 -> M1/M2/M3 关系 | 白沙墓群不是孤立墓葬 |
| 墓门 | 门额、门额背面彩画、封门砖 | 构件位置 -> 彩画位置 -> 封闭方式 | 墓门既是入口，也是第一道证据界面 |
| 甬道 | 东西壁、顶部、叠胜彩画 | 两壁对照 -> 顶部节奏 -> 通道转换 | 甬道承担过渡与引导 |
| 前室 | 女乐、器物、壁函、北壁分区、顶部铺作 | 礼仪图像 -> 器物组合 -> 空间秩序 | 前室是礼仪展示空间 |
| 过道 | 纪年题记、破子棂窗、宝盖 | 文字 -> 建筑图像 -> 顶部结构 | 过道是时间信息和空间转换的交汇点 |
| 后室一版 | 妇人启门、地券、人骨铁钉、分布图 | 图像 -> 文书 -> 遗存位置 | 后室集中墓主身份与葬仪信息 |
| 后室二版 | 镜台、曲足盆架、杌、高几、灯菜、剪刀熨斗、顶部小铺作 | 日常器物 -> 墙面分区 -> 顶部结构 | 后室形成日常化叙事 |
| 终章 | 所有结论卡 | 空间 + 图像 + 文书 + 遗物 | M1 的意义来自多层材料共同指向 |

## 五、NPC 叙事方式

玩家身份：考古实习生。

NPC 只在章节入口、小游戏前后、关键结论汇总时出现，不介入每个普通热区。

| NPC | 出现阶段 | 功能 | 说话风格 |
|---|---|---|---|
| 附近村民 | 墓外、未进入墓室前 | 提供地方记忆、墓群位置、传闻氛围 | 口语、简短、带不确定性 |
| 考古技工 | 墓门、甬道、结构观察 | 提醒砖缝、封门、构件、残损、施工痕迹 | 实操经验强，少理论 |
| 考古队员 | 各章节探索中 | 提醒拍照、编号、记录、对照图 | 像同伴协作，不直接给答案 |
| 考古领队（教授） | 章节收束、推理汇总 | 引导玩家把线索连成结论 | 冷静、学术、克制 |

示例语气：

- 附近村民：“这片地早年有人挖出过砖，没人敢往深处动。”
- 考古技工：“先别急着看图案，砖缝和封门方式也要记。”
- 考古队员：“这张近景可以和东壁那张对照一下。”
- 教授：“单条线索不能说明问题，把题记、地券和遗存位置放在一起看。”

## 六、可加入的小游戏

小游戏应当是“考古工作动作”的游戏化，而不是无关机关。

| 小游戏 | 放置位置 | 玩法 | 奖励线索 |
|---|---|---|---|
| 平剖面定位 | 墓外 / 入墓前 | 把墓门、甬道、前室、过道、后室放回正确位置 | 解锁定位导航图 |
| 封门砖观察 | 墓门 | 按顺序点出门额、封门砖、门洞 | 解锁墓门结论 |
| 顶部彩画拼合 | 甬道 | 对齐叠胜彩画和顶部边线 | 解锁甬道顶部线索 |
| 前室礼仪归类 | 前室 | 把女乐、器物、壁函、铺作归入功能组 | 解锁礼仪空间结论 |
| 题记辨读 | 过道 | 点选题记关键字或行列 | 解锁年代链 |
| 遗物定位 | 后室 | 根据分布图放置地券、人骨、铁钉 | 解锁后室遗存关系 |
| 日常器物归类 | 后室第二版 | 镜台、盆架、杌、高几、剪刀熨斗归类 | 解锁日常化叙事结论 |
| 终章线索墙 | 结尾 | 把线索卡连接成推理网 | 解锁最终研究报告 |
