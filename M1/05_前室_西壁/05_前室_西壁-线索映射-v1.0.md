## 二、二级细节清单

### L05（M1 前室西壁 砖雕注子与壁画高瓶的年代矛盾）
- 二级细节 1：注子砖雕与壁面砖缝对齐，为建墓时同期嵌入；高瓶壁画覆盖了一道更老的墨线轮廓。
- 二级细节 2：砖砌桌面的侧面露出三道平行的细槽，槽内积灰比桌面其他处更厚。
- 二级细节 3：高瓶阴影方向指向左侧，与壁画中袍服的褶皱光影方向相反。

## 三、一级细节（首次检查触发）

### L05 · 前室西壁砖雕注子与壁画高瓶的年代矛盾

前室西壁下部，砖雕注子嵌于壁面，实测高四寸，流口微翘，腹部圆鼓，为北宋中期短流注子形制。手电上移照向壁画，案上高瓶立于瓶座之上，瓶座下部刻三层莲瓣纹，每层八瓣，瓣尖外卷。细量瓶座总高，与注子相近，但莲瓣纹雕刻深度达三分，远超砖雕注子的浮雕深度。壁画颜料层下的白灰有二次涂抹痕迹，厚度比注子周围原始白灰层多出一分。

## 九、实施建议

### 单区域Demo可行范围

本区域直接支撑 **L05**，是前室最核心的年代学矛盾线索：

1. **核心发现**：玩家在同一面壁上发现北宋中期注子与晚期高瓶瓶座共存
2. **关键证据**：砖雕注子与壁面砖缝对齐（同期嵌入）vs 高瓶壁画覆盖老墨线轮廓（后期绘制）
3. **测量验证**：瓶座莲瓣纹雕刻深度三分 vs 注子浮雕深度，形成可量化的年代差异证据
4. **关联触发**：L05 与 L07（墓道与墓门）的 R01 关联可在此区域初步建立——两处二次绘制使用相同批次的细砂白灰

**核心解谜链**：发现注子 → 测量形制 → 发现高瓶 → 对比年代 → 推断二次绘制 → 触发 R01
