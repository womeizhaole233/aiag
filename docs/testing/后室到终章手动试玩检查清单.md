# 后室到终章手动试玩检查清单

适用分支：`feature/scene-art-m1`

用途：手动检查后室证据链、待验证降级、后室组合判断、终章汇总入口是否顺序清楚，且不会提前解锁。

## 0. 试玩准备

- [ ] 打开游戏首页：`index.html`。
- [ ] 使用一个能进入后室的存档，或从前置章节正常推进到后室。
- [ ] 打开记录夹，确认能看到“观察记录 / 工具复查 / 待验证 / 已排除”等分区。
- [ ] 若使用旧存档测试，先确认记录夹中没有已经生成的 `后室组合判断` 和 `终章汇总结论卡`，否则会跳过部分锁定状态。

## 1. 后室入口与终章入口锁定

- [ ] 进入后室入口总览。
- [ ] 在未完成后室汇总前，尝试点击“查看终章汇总”入口。
- [ ] 预期：终章汇总不能查看。
- [ ] 预期提示应说明：需要先完成五个章节的组合判断；后室需要完成复查、降级和“汇总后室”。
- [ ] 预期缺失列表中至少应包含：后室组合判断、后室汇总。
- [ ] 如果只完成后室但前面章节没有完成，终章仍不应提前解锁。

## 2. 后室葬具证据链

- [ ] 打开记录夹，查看“葬具证据链复查”进度卡。
- [ ] 按小面板顺序收集：
  - [ ] 北壁 -> 北壁下部：点砖床边界、假门-砖床轴线。
  - [ ] 北壁 -> 人骨与铁钉：点人骨与铁钉、铁钉数量。
  - [ ] 北壁 -> 出土物分布图 -> 人骨位置：点分布图和人骨位置。
  - [ ] 北壁 -> 地券 -> 地券并盖：点地券本体。
- [ ] 预期：证据未收齐时，复查按钮不可点，并显示还缺几项证据。
- [ ] 预期：7 条证据收齐后，按钮变为“串联葬具证据链”。
- [ ] 点击后应生成记录：`葬具证据链复查`。

## 3. 待验证降级

- [ ] 打开记录夹的“待验证”分区。
- [ ] 检查妇人手部断口：
  - [ ] 未完成假门图像复查时，应提示先完成假门图像复查。
  - [ ] 完成后，应出现“将妇人手部新断口降级为局部残损”按钮。
  - [ ] 点击后，该记录进入“已排除”。
- [ ] 检查铁钉数量异常：
  - [ ] 未完成葬具证据链复查时，应提示先完成葬具证据链复查。
  - [ ] 完成后，应出现“将铁钉数量异常降级为角点细节”按钮。
  - [ ] 点击后，该记录进入“已排除”。
- [ ] 预期：可处理的待验证记录会排在前面。

## 4. 后室组合判断

- [ ] 打开记录夹的“工具复查”分区。
- [ ] 查看“后室组合判断”进度卡。
- [ ] 预期清单包含 5 项：
  - [ ] 假门图像复查。
  - [ ] 地券文书复查。
  - [ ] 葬具证据链复查。
  - [ ] 降级手部断口。
  - [ ] 降级铁钉数量。
- [ ] 未完成任意条件时，按钮应显示“还缺 X 项条件”，不可生成。
- [ ] 三条复查完成但两条降级未完成时，应显示还缺 2 条降级。
- [ ] 五项全部完成后，按钮应变为“形成后室章节组合判断”。
- [ ] 点击后应生成记录：`后室组合判断`。

## 5. 汇总后室

- [ ] 回到后室入口总览，点击“汇总后室”。
- [ ] 若后室组合判断还没有生成，预期仍被拦住。
- [ ] 若后室组合判断已生成，预期后室章节完成。
- [ ] 完成后应出现后室章节完成提示，并自动推动终章结论卡生成条件检查。

## 6. 终章汇总入口

- [ ] 后室完成后，再点击“查看终章汇总”。
- [ ] 如果墓门、甬道、前室、过道中任意章节未完成，预期仍不能查看，并列出对应缺失章节。
- [ ] 如果五个章节都完成，预期终章汇总可查看。
- [ ] 解锁后提示应说明：五个章节判断已经可以并读，终章结论卡已经在线索墙中生成。
- [ ] 打开“结论”面板，确认能看到 `终章汇总结论卡`。

## 7. 美工与场景显示重点

- [ ] 移动端打开记录夹时，后室葬具证据链小面板不应横向溢出。
- [ ] “后室组合判断”5 项清单应能完整滚动查看。
- [ ] “查看终章汇总”和“汇总后室”两个入口在后室入口总览上不应互相遮挡。
- [ ] 终章入口未解锁时，提示文字不应被弹窗边界裁切。

## 8. 通过标准

- [ ] 后室复查、降级、组合、汇总四层顺序清楚。
- [ ] 玩家知道还缺哪些复查和降级。
- [ ] 终章不会在五个章节完成前提前解锁。
- [ ] 终章解锁后，文案不再像占位，而是指向已生成的终章结论卡。
