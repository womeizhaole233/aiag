# M1第3步美工视图优化检查报告

更新时间：2026-06-08  
适用分支：`feature/scene-art-m1`  
检查对象：当前 `data/scenes.js` 中所有 scene view，重点检查第 1、2 步新增视图，以及后室、前室细节图。

## 1. 检查结论

当前场景数据结构稳定：

- 场景数：`6`
- 当前视图数：`72`
- 新增视图数：`25`
- `assets/M1` 图片数：`69`
- `data/scenes.js` 引用图片数：`69`
- 缺失图片：`0`
- 未使用图片：`0`
- 跳转断链：`0`

主要问题不是图片缺失，而是美工呈现方式：

1. 部分细节图尺寸太小或比例过窄，不适合直接作为整屏背景。
2. 后室新增证据图质量较好，但多数热点目前还是整图大热点，需要进一步细化为对象级热点。
3. 后室西南壁仍是当前最大结构问题：`rear_southwest` 和 `rear_southwest_mirror_closeup` 仍使用同一张镜台图，缺少真正的西南壁总览或组合图。
4. 前室新增人物/仪仗细节图需要改成局部放大、嵌入式信息图或放大镜 UI，而不是普通大场景视图。

## 2. P0 优先处理项

### 2.1 后室西南壁需要组合图或替换图

涉及视图：

- `rear_chamber/rear_southwest`
- `rear_chamber/rear_southwest_mirror_closeup`

当前问题：

- 两个视图都使用 `第一号墓后室西南壁壁画中的镜台.png`。
- `rear_southwest` 名义上是西南壁总览，但实际只是镜台局部。
- 玩家会误以为镜台就是整面西南壁，曲足盆架、杌与镜台之间的空间关系不清楚。

建议处理：

- 优先制作一张“后室西南壁陈设组合图”。
- 图中至少包含镜台、曲足盆架、杌三个对象的相对位置。
- 如果没有真实整壁原图，可用现有三张局部图做学术标注型组合图。
- `rear_southwest` 改用组合图；`rear_southwest_mirror_closeup` 保留镜台近景。

建议美工类型：

- 组合图
- 信息图
- 半复原示意图

### 2.2 前室骨朵图不适合作整屏背景

涉及视图：

- `front_chamber/front_south_mace_closeup`

当前图片：

- `228x1075`
- 极窄竖图，横向信息很少。

当前问题：

- 作为整屏场景时左右空白会非常多。
- 热点覆盖面积过大，本质上只是“整件器物”识别。

建议处理：

- 不建议作为整屏背景。
- 改成前室南壁里的局部放大浮窗。
- 或制作“南壁仪仗细节组合图”，把骨朵与南壁位置关系放在同一张图中。

建议美工类型：

- 竖向局部放大镜
- 嵌入式细节卡
- 与南壁总图的定位连线

### 2.3 前室排箫图需要放大镜或重绘

涉及视图：

- `front_chamber/front_east_panpipe_closeup`

当前图片：

- `616x724`
- 分辨率偏低。

当前问题：

- 可以看清排箫，但不适合作为大背景。
- 如果强行铺满，质感会比莲花冠、东壁主图弱。

建议处理：

- 保留为局部细节图，但用放大镜 UI 呈现。
- 可做轻度超分、手绘增强或描线重绘。
- 与 `front_east_lotus_crown_closeup`、`front_east_shoes_closeup` 组成“女乐细节组”。

建议美工类型：

- 局部放大
- 描线增强
- 女乐细节组合图

### 2.4 甬道酒瓶图需要降级为小细节

涉及视图：

- `corridor/corridor_wine_bottle_closeup`

当前图片：

- `452x820`
- 小尺寸竖图。

当前问题：

- 不适合作整屏背景。
- 适合作为 H/P1 级细节，不应抢过甬道顶部、东壁、西壁主线。

建议处理：

- 改为局部放大浮层。
- 或在甬道西壁图上用放大镜标出位置，再弹出酒瓶细节。

建议美工类型：

- 小型证据浮窗
- 放大镜局部

## 3. P1 优先处理项

### 3.1 后室新增证据图需要细化热点

涉及视图：

- `rear_false_door_diagram`
- `rear_north_lower_closeup`
- `rear_southeast`
- `rear_southeast_detail`
- `rear_land_deed_cover_closeup`
- `rear_porcelain_bowl_closeup`
- `rear_bones_position_closeup`

当前优点：

- 图片尺寸基本足够。
- 适合承载后室证据链。
- 可作为后室结构、葬具、器物、东南壁补充的正式图。

当前问题：

- 目前多数只有一个大面积观察热点和一个返回热点。
- 热点面积普遍超过整图 `0.5`，更像临时占位。

建议处理：

- `rear_false_door_diagram`：拆为门框、门扇、门缝、妇人位置。
- `rear_north_lower_closeup`：拆为假门下沿、砖床关系、下部边界。
- `rear_southeast`：拆为人物组、桌案/器物、壁面边界。
- `rear_southeast_detail`：拆为人物、桌面、器物细节。
- `rear_land_deed_cover_closeup`：拆为地券本体、盖、比例标尺。
- `rear_porcelain_bowl_closeup`：拆为碗片、釉色、出土背景。
- `rear_bones_position_closeup`：拆为头骨、肢骨、葬具边界、方向标。

建议美工类型：

- 对象级热点
- 证据编号标注
- 局部辅助线

### 3.2 后室北壁热点开始拥挤

涉及视图：

- `rear_chamber/rear_north`

当前问题：

- 北壁主图现在承担太多入口：妇人启门、地券、人骨、分布图、假门结构图、北壁下部。
- 下方区域热点较密，容易视觉拥挤。

建议处理：

- 将北壁主图改成“主对象 + 复查入口”两层。
- 把地券、人骨、分布图、假门结构图放进一个“后室证据复查”侧边菜单或图标组。
- 保持北壁正图的视觉重点在假门与妇人启门。

建议美工类型：

- 右侧证据索引
- 图标组
- 复查面板

### 3.3 前室新增细节需要分组

涉及视图：

- `front_east_lotus_crown_closeup`
- `front_east_panpipe_closeup`
- `front_south_mace_closeup`
- 既有 `front_east_shoes_closeup`

当前问题：

- 莲花冠质量较好，可以独立展示。
- 排箫、骨朵、尖鞋更适合组成“前室人物/仪仗细节组”。
- 单独做成多个整屏视图会让体验碎片化。

建议处理：

- 做一个“前室人物与仪仗细节组合图”。
- 以东壁女乐为主，旁边放莲花冠、排箫、尖鞋；南壁骨朵作为仪仗补充。
- 玩家从前室东壁或南壁进入这个组合图，再分别点击细节。

建议美工类型：

- 组合图
- 证据卡组
- 局部放大镜

## 4. P2 优化项

### 4.1 后室东南壁可以成为正式侧壁视图

涉及视图：

- `rear_southeast`
- `rear_southeast_detail`

判断：

- 图片尺寸和观感都较好。
- 可以保留为正式后室侧壁视图。

建议：

- 下一步只需要做热点细化和轻微裁切。
- 不需要重绘。

### 4.2 地券并盖、白瓷碗、人骨位置适合证据链 UI

涉及视图：

- `rear_land_deed_cover_closeup`
- `rear_porcelain_bowl_closeup`
- `rear_bones_position_closeup`

判断：

- 三张图适合一起构成“后室葬具证据链”。
- 不建议作为互不关联的三个孤立视图。

建议：

- 做一个证据链 UI：地券并盖 -> 白瓷碗 -> 人骨位置 -> 出土物分布图。
- 每张图保留可点击细节，但视觉上要有连线或编号。

### 4.3 墓门和墓外图暂不急于重绘

涉及视图：

- `environment_baisha_context`
- `tomb_gate_structure_diagram`
- `tomb_gate_back_corridor`

判断：

- 图片尺寸足够。
- 当前主要是资料图/结构图，不是美术短板。

建议：

- 后续可加标签、箭头、区域框。
- 暂不作为本轮美工优化重点。

## 5. 推荐执行顺序

1. 制作 `rear_southwest` 后室西南壁组合图。
2. 将前室排箫、骨朵、甬道酒瓶改成局部放大/浮窗呈现。
3. 细化后室新增证据图热点，优先假门、东南壁、地券并盖、人骨位置。
4. 制作“前室人物与仪仗细节组合图”。
5. 制作“后室葬具证据链 UI”。
6. 统一所有细节视图的返回按钮、放大镜样式和证据编号样式。
