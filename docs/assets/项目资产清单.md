1. 当前运行关卡只引用 `assets/` 或 `assets/M1/` 中已经进入项目的图片。
2. 原始资料夹 `D:\考古游戏\图片-底图版\M1` 保持为母库，不直接在网页中引用。
3. 每扩展一个新区域，先从原始资料中挑选主底图、特写图、证据图，再复制到项目内。
4. 旧 `data/*.json` 暂不继续扩展，后续可以重建为更干净的 `scenes`、`hotspots`、`clues` 数据结构。
5. 根目录快捷图可以保留给当前原型，但正式开发时应以 `assets/M1` 的区域分类为主。
