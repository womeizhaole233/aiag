# M1 各章节线索母表：剧情深化版 v1.1

> 基于 `M1线索深化设计方案_剧情融合_v1.0.md`、`M1各章节线索母表_v1.0.md`、各章线索精修版与《线索文案规范.md》整理。
> 本版不是替换学术母表，而是在既有 `E/P0/P1/H/R` 线索上叠加剧情、复查和玩家成长字段。真实考古证据仍按 v1.0 母表为准；剧情虚构或半虚构内容应以 `F`、剧情增强字段、可推测或仍存疑处理，不直接混入可判断学术 P0。
> 阅读补充：本表中的“不能”主要指不能直接写入“可判断”学术结论，不等于禁止剧情负责人写人物猜测或悬念讨论。当前硬性禁止只有三条：题记一条线索解释整座墓；妇人启门、假门写成真实通道或密室入口；NPC 直接告诉玩家本章 R 结论。

## 一、v1.1 使用原则

| 原则 | 说明 |
| --- | --- |
| 学术证据不降格 | 原有 `E/P0/P1/H/R` 的学术功能保持不变，章节结论仍由可见事实、复查动作和组合判断支撑。 |
| 剧情内容不冒充证据 | 赵怀诚、银扣、暗格、反切、绢帛等剧情内容如缺少真实材料来源，应标为 `F`、剧情增强字段、可推测或仍存疑。 |
| H 线索先吸引后处理 | H 线索允许制造好奇心、误读和讨论，但进入结论前需要复查、降级或保留存疑。 |
| NPC 只做方法引导 | NPC 可以提示复查、质疑过度解释、解释知识卡，但不直接告诉玩家本章 R 结论。 |
| 终章保留存疑栏 | 终章不是破案，而是提交“可判断 / 尚不能判断 / 剧情可疑但证据不足”的三栏报告。 |

## 二、新增字段定义

| 字段 | 填写要求 |
| --- | --- |
| 剧情悬念功能 | 这条线索让玩家追问什么；只写问题方向，不写定论。 |
| 角色触发/反馈 | 谁最适合发现、解释、质疑或收束这条线索。 |
| 玩家操作玩法 | 玩家需要做的考古动作，如定位、拓片、侧光、辨读、归类、叠合、降级。 |
| 回收节点 | 这条线索在哪一章或终章被重新理解。 |
| 剧情边界 | 这条线索哪些内容需要分栏为可推测、仍存疑或待确认。 |
| 建议数据处理 | 进入 `observation/check/hypothesis/relation/F` 中哪一层，方便后续接代码。 |

## 七、玩家操作闭环

| 状态 | 含义 | UI 建议 | 可进入结论吗 |
| --- | --- | --- | --- |
| 已观察 | 玩家点击并记录了可见事实。 | 观察记录夹。 | 否，除 E 类定位外通常还需复查。 |
| 待复查 | 线索有价值但尚未使用工具或对照。 | 待复查栏，显示推荐工具。 | 否。 |
| 已复查 | 完成拓片、侧光、辨读、归类、叠合等动作。 | 复查记录栏。 | 可以进入假说或关系链。 |
| 待验证 | 可能是 H 或剧情 F，玩家有疑问但证据不足。 | 待验证栏。 | 不能直接进入学术结论。 |
| 已降级 | 复查后确认不能承担强解释。 | 已排除/已降级栏。 | 只可作为方法学习或边界说明。 |

| 存疑 | 终章仍不能判断的问题。 | 存疑栏。 | 可进入最终报告的“不能确定”部分。 |

## 八、章节 R 解锁强化条件

| 章节 | 原 R 条件 | v1.1 增强条件 |
| --- | --- | --- |
| 墓外 | 完成地图、平剖面、空间序列。 | 传闻必须完成分类；银扣只能进入个人线。 |
| 墓门 | 完成门额、彩画、封门结构。 | “永安”若出现，必须标存疑；砖缝 H 必须降级或标不能用于结论。 |
| 甬道 | 完成顶部、东壁、西壁三面合读。 | 至少一次把单点异常降级，证明玩家掌握对照方法。 |
| 前室 | 完成人物、器物、顶部归类。 | 尖鞋或单件器物必须降级；屏风隐字如出现，只能作为疑似文字进入剧情栏。 |
| 过道 | 完成题记、窗格、顶部复查。 | 题记过度解释必须被降级；“赵大翁”只能作为身份悬念挂起。 |

| 终章 | 完成空间、图像、文书遗存链。 | 必须提交“可判断 / 可推测 / 仍存疑”三栏报告。 |

## 十一、一句话落地目标

v1.1 的目标不是给 M1 添加更多“神秘线索”，而是让每个神秘感都必须经过考古动作：先观察，再复查，再组合，最后把能判断的写入结论，把不能判断的诚实留在存疑栏。
