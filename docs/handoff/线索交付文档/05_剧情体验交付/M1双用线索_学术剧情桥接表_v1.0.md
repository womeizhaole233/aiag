用途：给剧情、关卡、交互与美术团队标出“既有学术证据功能，又能推动主线剧情”的双用线索。

## 0. 使用原则

1. P0/P1线索可以承担剧情触发、情绪推进和终章回收功能，但不得改写其学术结论。
2. F线索只作为剧情内容或互动包装，不得冒充考古证据，也不得单独支撑终章学术判断。
3. 玩家最终应能区分三类结论：可判断、可推测、仍存疑。
4. 主线剧情节点优先选择“证据稳定、情绪明确、可二次回收”的线索。

## 3. 每个场景的剧情钩子小结

### 3.1 墓外/全景

剧情钩子：玩家先获得一座完整墓葬的空间轮廓，知道自己接下来不是在找“单个谜题”，而是在逐层进入一个真实墓室系统。
适合情绪：开场的探索欲和职业感。
边界：墓外和平剖面只提供空间骨架，不提供人物动机。

### 3.2 墓门

剧情钩子：墓门最适合承担“进入”和“复查”的双重体验。门额背面彩画让玩家意识到正面信息不够，封门砖则提供进入墓室前的仪式边界。
适合情绪：谨慎、兴奋、第一次发现隐藏信息。
边界：“永安”等剧情内容可以挂在这里，但必须标F或待核，不能变成墓门考古结论。

### 3.3 甬道

剧情钩子：甬道的重点不是大反转，而是让玩家学会考古观察方法：顶部、侧壁、背面要合读。
适合情绪：玩家开始觉得自己“会看”了。
边界：叠胜彩画与甬道图像不能被解释成秘文、机关或后期异常。

### 3.4 前室

剧情钩子：前室展示礼仪、乐舞、宴饮和秩序，是剧情里的“表面世界”。它越完整、越热闹，后室的人骨和文书出现时越有反差。
适合情绪：先被热闹吸引，再产生不安。
边界：前室壁画不能直接变成真实宴会记录，也不能指认具体人物情感。

### 3.5 过道

剧情钩子：过道是主线从“看墓”转向“追问人”的关键。题记给出纪年和“赵大翁”，玩家第一次抓住一个可追问的名字。
适合情绪：突然贴近历史人物。
边界：题记是P0时间锚点，不是后期添加、异常笔迹或伪造文本。

### 3.6 后室

剧情钩子：后室是M1主线情绪和证据最密集的场景。砖床与分布建立现场，人骨带来真实逝者，铁钉制造可复查疑问，地券提供文书对读，妇人启门则把图像悬念连接到F剧情。
适合情绪：沉重、惊奇、想读懂、最后回到克制判断。
边界：后室可以最有戏，但越有戏越要守边界；不得推出身份抹除、真实通道、风水诅咒、谋杀或后期进入。

### 3.7 终章

剧情钩子：终章不是揭开一个夸张阴谋，而是让玩家完成“我知道什么、我可以推测什么、我必须承认不知道什么”。这会让学术严谨本身成为游戏的情绪回报。
适合情绪：清醒、释然、完成感。
边界：F剧情可以被情感回收，但不能反向改写P0/P1结论。

## 4. F线索使用边界

以下类型线索可以服务剧情，但不应进入“学术证据功能”栏：

| 类型 | 可用方式 | 禁止方式 |
|---|---|---|
| 暗格、绢帛、私人文书 | 作为人物情绪、主线悬念、任务奖励或终章F栏回收。 | 作为墓主身份、死因、迁葬原因的学术证据。 |
| “永安”等剧情拓片/题字 | 作为玩家追问和NPC对话触发点。 | 作为真实铭文结论、诅咒证据或后期进入证据。 |
| 银扣、梦境、家族记忆 | 作为开场动机、角色关系或玩家代入。 | 作为考古现场证据。 |
| 风水、诅咒、身份抹除等强叙事 | 只能作为角色误读、民间传说或红鲱鱼。 | 作为终章核心真相。 |

## 5. 推荐落地方式

1. 主线剧情优先绑定S级线索：PASS-P0-01、HS-P0-02、HS-P0-04、HS-P0-05。
2. A级线索用于前置、复查和情绪转折：GATE-P0-02、GATE-P0-03、FRONT-P0-02、HS-P0-03、END-P0-03。
3. F线索可以增强快乐、悬念和人物温度，但界面上应明确进入“剧情记录”或“待核记录”，不要进入“考古结论”。
4. 每个主线节点都建议配一个“证据边界提示”：玩家可以感到好奇，但系统要教会玩家为什么不能下过度结论。
