# 一、过道

## 1. 场景目标

过道是玩家从前室进入后室前的最后一次“刹车”。
这一场景要完成三件事：

1. 让玩家第一次抓住一个可追问的名字：`赵大翁`。
2. 让玩家意识到文字很重要，但不能只靠文字下结论。
3. 用窗格和顶部结构，把“通往后室”的悬念建立起来，但不把它写成暗道或机关。

## 2. 玩家情绪目标

从“刚离开前室的热闹”转入“窄道里的突然贴近”。
玩家应先感到：这里有字，这里终于有人名了。
随后被系统拉回：慢一点，文字只是锚点，还要看墙、窗、顶部和方向。

情绪曲线：
收窄 -> 发现题记 -> 兴奋追问 -> 复查降温 -> 带着疑问进入后室。

## 10. 本场景结论

过道不是单纯的窄路。
它同时提供：

- 题记带来的 1099 年时间锚点和“赵大翁”称谓；
- 破子棂窗带来的仿木窗格表达；
- 顶部宝盖带来的轴线和空间转折；
- 东壁图像带来的导向补充。

本场景只能收束为：过道集中出现时间信息与空间转换信息，是进入后室前的关键转换节点。

## 11. 留到终章的悬念

1. 赵大翁是谁？是否与赵怀诚剧情线有关？
2. 1099 年题记与后续文书/绢帛中的时间信息如何对读？
3. 像窗、像门的图像，会不会在后室继续出现？

终章边界：
1099 年和“赵大翁”称谓可进入可判断；赵大翁与赵怀诚关系只能进入可推测或仍存疑。

# 二、后室

## 1. 场景目标

后室是 M1 的情绪高潮和证据密集区。
这一场景要让玩家强烈感觉到：终于到了最深处，但越像答案现场，越不能急着宣布真相。

后室要完成四件事：

1. 用妇人启门和假门制造视觉悬念。
2. 用砖床、人骨、铁钉和地券把玩家拉回真实遗存。
3. 用日常器物图像建立“地下寝居”的温度。
4. 把暗格、反切素帛、赵怀诚绢帛和 1099/1106 时间差明确放入 F/待核剧情链。

## 2. 玩家情绪目标

从“门后是否有路”的好奇，转向“这里真的安葬过人”的沉重。
再从“是不是发生过什么”的惊疑，转向“我需要分清证据和故事”的克制。

情绪曲线：
悬念 -> 震动 -> 不安 -> 读文书 -> 共情 -> 克制收束。

| 2 | HS-E-02 | 后室多壁面 | 开启北壁、南壁、侧壁、顶部、出土记录复查。 |
| 3 | HS-P0-05 | 妇人启门与假门 | 先给视觉悬念，但随后降级真实通道误读。 |
| 4 | HS-P0-01 | 后室砖床 | 建立遗存定位基准。 |
| 5 | HS-P0-02 | 人骨位置 | 情绪核心，确认实际安葬遗存。 |
| 6 | HS-P0-03 | 铁钉与葬具痕迹 | 制造不安，再收束为葬具范围复查。 |
| 7 | HS-P0-04 | 地券与券盖 | 文书证据登场，连接题记和终章时间线。 |

| 11 | HS-R-01 | 后室章节结论 | 生成“安葬核心区 + 地下寝居图像”的章节判断。 |

## 10. 本场景结论

后室不是“密室真相现场”，也不是单纯的壁画空间。
它应收束为：

后室既是 M1 的安葬核心区，也是被图像塑造成地下寝居的空间。

可判断部分：

- 后室位于全墓序列末端；
- 砖床是遗存定位基准；
- 二具人骨支持实际安葬判断；
- 十九枚铁钉可辅助葬具范围复原；
- 地券、券盖、朱书券文属于葬仪文书线索；
- 妇人启门与假门属于图像空间延续母题；
- 多壁面器物图像补充地下寝居感。

不可在本场景坐实：

- 死因；
- 具体身份；
- 迁葬原因；
- 盗扰或二次开棺；
- 真实通道、密室或机关；
- 风水诅咒、身份抹除或案件真相。
