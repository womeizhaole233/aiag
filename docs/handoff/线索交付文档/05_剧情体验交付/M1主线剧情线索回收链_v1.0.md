## 0. 回收链使用原则

1. P0/P1 线索可以触发剧情疑问，但回收时需要区分“可判断 / 可推测 / 仍存疑”。
2. F 线索可以进入剧情记录、可推测栏或仍存疑栏，不直接覆盖 P0/P1 的可判断学术结论。
3. H 线索可以用于误读、争论、降级和审稿；进入结论前需要复查、降级或保留存疑。
4. 主线剧情的情绪可以越来越强，但终章表达必须越来越克制。

当前硬性禁止只有三条：

1. 把题记一条线索写成解释整座墓的答案。
2. 把妇人启门、假门写成真实通道或密室入口。
3. 让 NPC 直接告诉玩家本章 R 结论。

## 1. 总回收链

银扣个人动机 -> “永安”待核文字 -> 屏风隐字待核 -> 赵大翁称谓 -> 后室人骨/铁钉/地券/妇人启门 -> 暗格/反切素帛/赵怀诚绢帛 -> 1099/1106 时间差 -> 终章三栏报告。

这条链的核心体验是：玩家从“我为什么来到这里”进入“墓里是否有人留下话”，再从“赵大翁是谁”进入“真实遗存与剧情自述如何分栏”。最终不是证明一个唯一隐秘真相，而是把证据、推测和故事分清。
