## 一、总判断

现有深化方向是对的：应该让玩家通过定位、记录、拓片、测绘、侧光、辨字、叠合和存疑报告，亲手生成结论。

但现有文档中仍混入了一批早期“强悬疑灵感池”的设定。这些设定有游戏张力，但与 `白沙宋墓-简体版` 的学术条目、以及《白沙尘烟3》的最新剧情写法不完全一致。它们不能继续作为 M1 正式学术结论使用。

最重要的修正原则：

> 学术线要回到白沙一号墓真实证据；剧情线可以保留赵怀诚、暗格、反切和绢帛，但必须作为 `F / 剧情推测 / 可推测栏`，不能改写考古 P0。

## 五、现有深化结论修订版

### 学术最终结论

M1 是一座北宋元符二年左右的赵姓民间精英砖室墓。它具有墓道、墓门、甬道、前室、过道、后室的连续空间序列，墓门和墓室以砖仿木构模拟地上建筑，前室通过乐舞、宴饮、贡纳、侍卫和器物图像形成礼仪展示，过道题记提供年代与称谓锚点，后室以砖床、人骨、铁钉、地券和随葬器物构成安葬核心，并以妇人启门、假门和日常器物图像营造地下寝居。镇墓钱、金井/穴孔若后续接入，需另列为待核补充，不覆盖当前总表。

### 剧情阶段结论

在不冒充考古定论的前提下，赵怀诚剧情线可以这样收束：

> “赵怀诚”把姓名、罪责和辩白拆散藏在墓中。玩家通过屏风隐字、暗格铜筒、反切素帛、绢帛残字和时间线复查，把这份自述重新拼合出来。但它最终只能是墓中自述，不能直接等同官方史实。

### 玩家体验结论

玩家最后获得的不是“破案成功”，而是：

1. 会看地形、封门和空间序列；夯层、瓷片等内容若接入，需另列待核或后续扩展。
2. 会读壁画功能，而不是只看漂亮图像。
3. 会把题记、地券、人骨、铁钉、器物放在一起判断。
4. 会把剧情推测和学术结论分开。
5. 会在报告里写下“仍存疑”。

原因：这条提示词保留“快乐”，但把快乐建立在动手复查和边界判断上，而不是靠强悬疑硬推。

```text
请在不破坏白沙宋墓 M1 学术结论的前提下，检查赵怀诚剧情线是否正确分栏。银扣、“永安”、屏风隐字、暗格、反切素帛、绢帛自述、1099/1106 时间差等内容，只能标为 F、剧情增强、可推测或待核。

请逐条判断：
1. 它是否有真实考古材料支撑；
2. 若没有，是否已经标 F 或待核；
3. 它是否误导玩家把剧情自述当作官方史实；
4. 它是否覆盖了 P0/P1 学术结论；
5. 终章中应进入“可判断 / 可推测 / 仍存疑”哪一栏。

最后输出一份“剧情相关线索清单”和“不能写成学术定论的内容清单”。
```

原因：赵怀诚线可以作为情感发动机，但交付时必须和学术线隔开，才能既好玩又可靠。

## 八、一句话结论

现有深化最大的方向没错，但必须先把“强悬疑灵感池”从“正式学术证据链”里清出来。真正能让玩家快乐的，不是把墓写成阴谋现场，而是让玩家在一次次观察、动手、复查、被系统或角色反馈轻轻纠正之后，发现自己真的变得更会看一座墓。
