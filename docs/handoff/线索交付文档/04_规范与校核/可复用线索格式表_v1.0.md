# 可复用线索格式表 v1.0

> 适用范围：白沙宋墓 M1 各章节线索设计，包括墓外、墓门、甬道、前室、过道、后室与终章。
>
> 核心原则：点击弹窗只写玩家看到的现场事实；复查反馈才给出谨慎判断；章节结论必须由 P0/P1/E 线索支撑；H 线索只能用于误导排除或降级。

## 一、线索总表字段

| 字段 | 填写内容 | 写作要求 |
|---|---|---|
| 线索ID | 如 `HS-P0-01`、`FS-P1-03` | 章节缩写 + 层级 + 编号，保持唯一 |
| 章节 | 后室 / 前室 / 甬道等 | 对应 PRD 章节 |
| 场景/图片 | 图片文件名或场景名 | 尽量对应具体图片或视角 |
| 热点对象 | 玩家点击的位置 | 如砖床、地券、妇人手部、镜台 |
| 线索层级 | `P0/P1/P2/H/E/R/F` | 严格按 PRD 分类 |
| 内部设计说明 | 给策划/开发看的功能说明 | 可以写学术作用、关卡作用和推理价值，不直接给玩家看 |
| 点击弹窗文案 | 玩家第一次点击看到的文字 | 控制在 1-2 句，只写可见事实和现场感，不直接讲学术结论 |
| 记录夹文案 | 进入记录夹的短文案 | 比弹窗更简洁，方便复查 |
| 复查动作 | 玩家后续要做什么 | 如标尺、放大、分布图定位、双图对比、归类 |
| 复查反馈 | 玩家复查后得到的谨慎判断 | 可以出现初步解释，但不能把可能性写成事实 |
| 小结论连接 | 这条线索支撑哪一句章节结论 | 必须能连接到章节 R 结论 |
| 关联线索 | 可组合的其他线索ID | 如 `HS-P0-02 + HS-P0-03` |
| 不能过度推断 | 明确禁止推出什么 | 防止盗墓、谋杀、身份、密室等跳跃推理 |
| 是否进记录夹 | 是 / 否 | P0/P1/H/E 一般可进 |
| 是否影响解锁 | 是 / 否 | P0 通常影响，P1 辅助，H 用于降级 |
| 备注 | 资料来源、待补图片、坐标问题 | 内部使用 |

## 二、单条线索填写模板

```markdown
### 【线索ID】线索名称

**章节**
后室

**场景/图片**
填写图片文件名或场景名

**热点对象**
玩家点击的位置

**线索层级**
P0 / P1 / P2 / H / E / R / F

**内部设计说明**
说明这条线索在推理链中的作用。这里可以写学术意义、关卡作用、为什么它重要。

**点击弹窗文案**
玩家第一次点击后看到的自然观察。控制在 1-2 句，只写现场可见内容，不直接写“这说明……”

**记录夹文案**
更短的记录版本。用于玩家之后复查。

**复查动作**
玩家需要进行的动作：放大、标尺测量、分布图定位、双图对比、器物归类等。

**复查反馈**
玩家完成复查后得到的谨慎判断。可以有初步解释，但不能把可能性写成事实。

**小结论连接**
说明它支撑哪一个章节小结论。

**关联线索**
列出可以组合判断的线索ID。

**不能过度推断**
明确写出这条线索不能推出什么。

**是否进记录夹**
是 / 否

**是否影响解锁**
是 / 否

**备注**
资料来源、图片问题、坐标问题或后续补充。
```

## 四、层级使用规则

| 层级 | 用途 | 文案重点 |
|---|---|---|
| P0 | 支撑章节小结论的核心证据 | 必须可见、可复查、可组合 |
| P1 | 解释 P0 的辅助证据 | 帮玩家理解空间、器物、图像意义 |
| P2 | 沉浸观察 | 增加现场感，不承担主要推理 |
| H | 误导/待验证 | 只能用于排除、降级、训练谨慎判断 |
| E | 环境/定位 | 帮玩家理解方位、空间、进入顺序 |
| R | 推理关系/章节结论 | 由复查和组合生成，不是普通点击线索 |
| F | 剧情虚构 | 必须和考古证据分开，不能支撑学术结论 |

## 五、写作检查清单

1. 点击弹窗是否只写“玩家看见了什么”？
2. 复查反馈是否只写“这些事实可以谨慎说明什么”？
3. 小结论连接是否明确服务于章节 R 结论？
4. H 线索是否只用于误导排除或降级？
5. 是否避免把单个异常直接解释为盗墓、谋杀、身份真相、密室机关或超自然事件？
6. 是否区分了考古事实、学术推论和剧情虚构？

## 六、推荐工作流

1. 先确定章节小结论。
2. 反推需要哪些 P0 线索。
3. 用 P1 线索解释 P0 的空间、器物或图像意义。
4. 加入少量 H 线索，让玩家通过复查学会降级判断。
5. 用 E 线索保证玩家能理解位置、朝向和空间关系。
6. 最后生成 R 章节结论卡。

## 七、最关键规范

点击弹窗写“玩家看见了什么”；复查反馈写“这些事实可以谨慎说明什么”；小结论连接写“它服务于哪条章节判断”；不能过度推断写“它绝对不能证明什么”。
