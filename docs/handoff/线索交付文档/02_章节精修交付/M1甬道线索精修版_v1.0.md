## 一、章节定位

| 项目 | 内容 |
| --- | --- |
| 核心学术问题 | 甬道如何通过顶部彩画、东壁人物和西壁人物/持物共同承担“过渡与引导”功能？ |
| 章节小结论 | 甬道不是普通通道。它位于墓门与前室之间，顶部叠胜彩画提供方向感，东壁与西壁人物图像分别组织守门、贡纳、交通和酒物等内容，共同建立从入口进入前室的图像秩序。 |
| 主线剧情可用方向 | 由队员提醒玩家“别只往前走，要抬头看顶部，也要分左右壁记录”。剧情节奏从墓门结构观察转入壁画阅读。 |
| 小游戏可用复查方向 | 顶部彩画对齐、东西壁人物功能匹配、题字与器物配对、顶部偏移误读降级。 |
| 证据边界 | 本章只能支持“甬道承担过渡与引导”。不能把某个人物、题字、马、黑瓶或顶部偏移写成墓主身份、具体事件、机关暗示、盗扰证据或后期进入证据。 |

## 四、三面合读规则

| 合读面 | 主要作用 | 对应线索 | 玩家应做什么 |
| --- | --- | --- | --- |
| 顶部 | 提供通道方向和空间连续感 | COR-P0-01 | 对齐纹样、边线和前后方向。 |
| 东壁 | 提供守门与贡纳趋入图像 | COR-P0-02 | 按人物动作和持物分类。 |
| 西壁 | 提供守门、马、酒物和题字图像 | COR-P0-03、COR-P1-02 | 识别人物、马、黑瓶和题字关系。 |
| 三面合读 | 形成“过渡与引导”的章节判断 | COR-P1-01、COR-R-01 | 把顶部、东壁、西壁放入同一张复查表，不让单点细节单独下结论。 |

## 五、H 线索降级规则

| 项目 | 设计 |
| --- | --- |
| 触发 | 玩家点击顶部局部偏移、边线差异、色差或磨损，并尝试解释为重绘、破坏、暗号、机关或后期进入。 |
| 复查步骤 | 1. 放大观察差异位置；2. 与相邻纹样和通道边线对照；3. 与东壁、西壁线索检查是否形成稳定关系；4. 判断它是否影响“甬道过渡与引导”的章节结论。 |
| 成功降级反馈 | “这处差异可以记录，但它不能单独证明重绘或破坏。先把它归入顶部彩画的局部保存/绘制差异中。” |
| 失败 / 提示反馈 | “只看一个偏移点还不够。回到顶部整体方向，再和东壁、西壁一起看。” |
| 降级结果 | `COR-H-01` 从“待验证”改为“已降级”；不参与 R 结论，只作为谨慎判断训练。 |

## 六、R 结论解锁条件

| 条件 | 要求 |
| --- | --- |
| 必须获得 | `COR-E-01`、`COR-P0-01`、`COR-P0-02`、`COR-P0-03` |
| 必须复查 | `COR-P1-01` 三面合读复查必须完成 |
| 建议复查 | `COR-P1-02` 酒瓶与题字建议完成，用于增强西壁判断，但不单独卡关 |
| 必须处理 | `COR-H-01` 必须被降级，或在记录夹中明确标为“待验证，不能用于结论” |
| 生成结论 | `COR-R-01` 生成后，玩家获得“甬道章节结论卡” |
| 解锁下一章 | 结论卡生成后开放前室章节 |

## 七、给主线剧情组的直接用法

| 节点 | 可写内容 | 不可写内容 |
| --- | --- | --- |
| 进入甬道 | 队员提醒玩家空间变窄，要抬头看顶部，也要分左右壁记录。 | 不写危险机关、压迫陷阱或神秘通道。 |
| 观察顶部 | 写成顶部彩画沿通道展开，提示玩家补充顶部记录。 | 不写顶部偏移说明重绘、破坏或暗号。 |
| 观察东壁 | 写成门旁人物、持物者和趋入方向。 | 不把人物命名为具体历史人物或剧情角色。 |
| 观察西壁 | 写成人物、马、黑瓶和题字之间的图像关系。 | 不把题字解释为墓主姓名、案件线索或身份真相。 |
| 章节收束 | 教授强调“顶部、东壁、西壁三面合读，甬道才成为引导空间”。 | 不在甬道阶段解释墓内最终结论。 |

## 九、一页版收束语

甬道章的玩家任务不是“在通道里找异常”，而是学会第二种考古判断：把顶部、东壁、西壁同时放入记录夹。顶部给出通道方向，东壁组织守门与贡纳，西壁组织守门、马、酒物和题字。三面合读之后，甬道才能被谨慎理解为从墓门进入前室的过渡与引导空间。
