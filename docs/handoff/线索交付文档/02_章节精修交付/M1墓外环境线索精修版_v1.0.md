# M1 墓外环境线索精修版 v1.0

> 基于《M1各章节线索母表_v1.0.md》和已完成的墓门、甬道、前室、过道、后室、终章精修版标准整理。
> 本版只打磨线索本身，暂不做图片、热点、坐标或代码映射。

## 一、章节定位

| 项目 | 内容 |
| --- | --- |
| 核心学术问题 | 玩家进入 M1 前，如何先建立墓群环境、地形范围、平剖面和空间序列入口，而不是从传闻或地形氛围直接推断墓主身份、年代细节或墓内事件？ |
| 章节小结论 | 墓外/环境章只负责建立调查背景与进入顺序：白沙所在区域、墓葬地形范围和平剖面共同说明玩家将进入的是一座具有连续空间序列的 M1。 |
| 主线剧情可用方向 | 开场可以有村民、队员和教授的低强度互动。村民提供地方记忆和发现线索，队员完成定位，教授收束为“先确认位置和顺序，再进入墓门”。 |
| 小游戏可用复查方向 | 地图定位、调查起点标注、平剖面空间排序、主轴线标注、传闻/证据分类、墓道初清前置确认。 |
| 证据边界 | 本章不能推出墓主身份、具体年代、墓葬等级、风水吉凶、盗扰、墓内事件或后室结论。墓外阶段的所有判断都应服务于“如何进入并记录 M1”。 |

| 条件 | 要求 |

| 解锁下一章 | 结论卡生成后，开放墓道初清或墓门章节。 |

## 七、给主线剧情组的直接用法

| 节点 | 可写内容 | 不可写内容 |
| --- | --- | --- |
| 开场定位 | 村民或队员带出白沙位置，玩家先看地图和地形。 | 不开场就宣布墓主身份、年代细节或墓内事件。 |
| 村民传闻 | 村民语气可以犹豫、口语化，提供地方记忆和发现线索。 | 不把传闻写成诅咒、灵异、盗墓线索或案件真相。 |
| 地形复查 | 队员提醒“先把墓群位置和调查范围标出来”。 | 不把地形直接写成风水吉凶或特殊选址结论。 |
| 平剖面排序 | 教授要求玩家按顺序确认墓道、墓门、甬道、前室、过道、后室。 | 不把空间顺序写成机关路线或隐藏暗号。 |
| 朝向记录 | NPC 说明主轴略偏，需要记录，但先不解释。 | 不把北偏东 15 度直接推成身份、避凶或异常。 |
| 章节收束 | 教授说“先建立位置，下一步看墓门”。 | 不在墓外阶段提前解释门额、壁画、人骨、地券或后室结论。 |

## 九、一页版收束语

墓外/环境章的任务不是解释 M1，而是建立进入 M1 的可靠前提。白沙附近图提供区域背景，白沙宋墓地形图确认墓群环境和调查范围，平剖面把墓道、墓门、甬道、前室、过道、后室连成连续空间序列，主轴线记录补充测绘意识。村民传闻可以制造开场氛围，但必须降级为背景信息，不能成为考古证据。完成本章后，玩家只应得到一个谨慎结论：M1 应作为墓群环境中的完整空间序列来进入；墓主身份、具体年代、墓内事件、盗扰、风水吉凶和后室结论都必须留到后续章节用实物、图像、文书和遗存继续复查。
