## 一、章节定位

| 项目 | 内容 |
| --- | --- |
| 核心学术问题 | 过道如何在前室与后室之间，同时提供文字时间锚点、仿木窗格和顶部结构转换信息？ |
| 章节小结论 | 过道不是单纯的窄路。纪年题记提供可复查的文字信息，破子棂窗体现仿木窗格意识，丁字盝顶式宝盖强化轴线与转折，东壁图像和空间位置共同把玩家从前室引向后室。 |
| 主线剧情可用方向 | 过道可作为“进入后室前最后一次复查”的章节。教授提醒玩家：文字很重要，但必须和结构、窗格、顶部、空间方向一起看。 |
| 小游戏可用复查方向 | 题记辨读、题记位置复查、窗格结构复原、顶部轴线对齐、流苏误读降级、题记过度解释降级。 |
| 证据边界 | 本章只能支持“过道集中出现时间信息与空间转换信息”。不能把题记、窗格、宝盖、流苏或东壁图像写成墓主身份真相、后室真相、机关路线、暗道、盗墓证据或墓内事件记录。 |

## 五、H 线索降级规则

| H 线索 | 触发 | 复查步骤 | 成功降级反馈 | 降级结果 |
| --- | --- | --- | --- | --- |
| PASS-H-01 流苏方向误读 | 玩家点击流苏并选择“方向/暗号/机关/暗道”类解释。 | 1. 回到同壁装饰整体；2. 对照流苏悬挂位置和构图；3. 检查是否与题记、窗格、顶部形成稳定关系；4. 若没有，归入装饰细节。 | “流苏方向值得记录，但它不能单独指示路线或机关。先归入装饰和构图观察。” | 从“待验证”改为“已降级”；不参与 PASS-R-01。 |
| PASS-H-02 题记过度解释 | 玩家完成题记辨读后，试图用题记解释墓主身份、后室结论或整墓事件。 | 1. 保留题记为时间锚点；2. 强制加入窗格、顶部、东壁图像复查；3. 检查题记能否单独支撑玩家结论；4. 若不能，降级其解释范围。 | “题记很重要，但它只能先作为时间锚点。进入后室前，还要和结构、图像和出土记录共同判断。” | 从“过度解释”降级为“时间锚点”；可参与 R，但不能单独生成 R。 |

## 六、R 结论解锁条件

| 条件 | 要求 |
| --- | --- |
| 必须获得 | `PASS-E-01`、`PASS-P0-01`、`PASS-P0-02`、`PASS-P0-03` |
| 必须复查 | 题记辨读、窗格复原、顶部轴线对齐三项都完成 |
| 建议复查 | `PASS-P1-01` 建议完成，用于增强题记位置判断；`PASS-P1-02` 建议完成，用于防止过道变成单一文字点 |
| 必须处理 | `PASS-H-01`、`PASS-H-02` 必须被降级，或在记录夹中明确标为“待验证，不能单独用于结论” |
| 生成结论 | `PASS-R-01` 生成后，玩家获得“过道章节结论卡” |
| 解锁下一章 | 结论卡生成后开放后室章节 |

## 七、给主线剧情组的直接用法

| 节点 | 可写内容 | 不可写内容 |
| --- | --- | --- |
| 进入过道 | 队员提醒玩家这是进入后室前的转换段，需要先复查文字和结构。 | 不写“通往密室”“机关通道”或危险封锁。 |
| 观察题记 | 写成可辨认的墨书题记，需要辨读和记录位置。 | 不把题记写成墓主身份真相、案件线索或后室答案。 |
| 观察窗格 | 写成仿木窗格在窄壁上展开。 | 不写窗后有真实空间、暗道或可开启机关。 |
| 观察顶部 | 写成顶部沿轴线收束，提示空间转折。 | 不把宝盖形制写成隐藏装置或密码。 |
| 观察流苏 | 写成容易误读的装饰细节，经复查降级。 | 不让流苏指路、指向暗门或预告机关。 |
| 章节收束 | 教授强调“文字是锚点，结构说明过道在转换”。 | 不在过道阶段解释后室遗存或终章大结论。 |

## 九、一页版收束语

过道章的玩家任务不是“读到题记就得到答案”，而是学会第四种考古判断：文字要和结构一起复查。纪年题记提供时间锚点，破子棂窗说明仿木窗格意识，顶部宝盖强化轴线和空间转折，东壁图像与流苏细节帮助玩家练习区分导向与误读。四组证据合读后，过道才能被谨慎理解为进入后室前的时间与空间转换节点。
