## 九、一页版收束语

前室章的玩家任务不是“在最漂亮的壁画里找身份答案”，而是学会第三种考古判断：把多壁面、多人物、多器物和顶部结构放进同一张复查表。东壁提供乐舞展示，西壁组织对坐与宴饮器物，南壁维持入口礼仪秩序，顶部提供仿木构建筑框架。只有四组证据合读，前室才能被谨慎理解为 M1 的礼仪展示空间与结构转折点。
