# M1 墓门线索精修版 v1.0

> 基于《M1各章节线索母表_v1.0.md》《可复用线索格式表_v1.0.md》《M1初步需求文档_PRD.md》和白沙宋墓知识材料整理。
> 本版只打磨线索本身，暂不做图片、热点、坐标或代码映射。

## 一、章节定位

| 项目 | 内容 |
| --- | --- |
| 核心学术问题 | 墓门如何同时作为正式入口、封闭边界和第一道结构/装饰证据界面？ |
| 章节小结论 | 墓门不是单纯“进入点”。门额正背观察、门额背面彩画、封门砖组织、门前地面和砖仿木构形式共同说明：玩家进入墓室前，已经面对一组需要复查的结构证据。 |
| 主线剧情可用方向 | 由考古技工引导玩家“先看门额、封门砖和下缘地面”。剧情语气应是现场清理与记录，不是破门探险。 |
| 小游戏可用复查方向 | 挖土消消乐后露出墓门；后续可做门额正背对照、封门砖层次观察、砖缝误读降级。 |
| 证据边界 | 本章只能支持“墓门是入口与证据界面”。不能推出盗墓、二次封门、藏匿、机关、密室或墓内事件真相。 |

| GATE-E-01 | E | 墓门外部位置 | 墓门位于墓道之后、甬道之前，是进入墓室的正式入口。 | 墓道尽头露出砖砌门面，门洞向内收束。这里是进入墓室前最明确的入口。 | 墓门位于墓道与甬道之间，是进入墓室的正式入口。 | 在平剖面或空间序列中标出“墓道-墓门-甬道”。 | 位置确认后，玩家知道墓门不是孤立画面，而是全墓序列中的入口节点。 | 支撑“墓门是正式入口”。 | 不能仅凭入口位置判断墓内保存状态、墓主身份或异常事件。 | 技工提醒“先确认门的位置，再看封门和门额”。 | 挖土消消乐完成后露出墓门区域；也可做空间排序确认。 |
| GATE-P0-01 | P0 | 墓门门额 | 墓门上方有横向门额构件，正面与背面是不同观察面。 | 门洞上方压着一条横向门额，边线和两侧门框可以分开辨认。 | 墓门门额需按正面、背面两个观察面记录。 | 建立“门额正面 / 门额背面”双栏记录，分别写位置、可见纹样和与门洞关系。 | 门额不是普通边框，它提供了进入墓门时最先需要对照的结构观察面。 | 支撑“墓门是第一道证据界面”。 | 不能把正背差异直接解释为后期藏匿、调换或改造。 | 队员可提醒“背面也要记，别只拍正面”。 | 可做“找出第二观察面”的门额对照任务。 |
| GATE-P0-02 | P0 | 门额背面彩画 | 门额背面可见卷草纹彩画；记录材料中可保留朱红底、墨线卷草、下缘裁切线等可见事实。 | 光扫到门额背面时，彩画纹样从暗处显出来。它不在第一眼能看到的正面。 | 门额背面保存卷草纹彩画，应与门额正面和门洞方向共同复查。 | 用侧光或放大观察背面彩画；只记录纹样、底色、边缘和所在观察面，再与 GATE-P0-01 对照。 | 背面彩画说明墓门保存了装饰信息，也说明门额需要正背复查；它本身不等于“被藏起来的秘密”。 | 支撑“墓门不仅是通行结构，也保存装饰证据”。 | 不能直接说彩画被故意隐藏、转置、涂改或用于传递暗号。 | 可制造“发现第二观察面”的小悬念，但教授要立刻收束为复查要求。 | 可作为门额正背对照的核心目标；成功后更新记录夹。 |
| GATE-P0-03 | P0 | 封门砖组织 | 墓门区域可见封门砖、砖层、灰缝和门洞边界关系。 | 门洞内有成层砖面和灰缝，砖层贴着门口收束。 | 封门砖与门洞、灰缝和门前地面共同构成封闭结构证据。 | 标出封门砖层、灰缝、门洞边界和下缘位置；与 GATE-P1-02 一起复查。 | 封门判断不能只看一块砖或一条缝，要看砖层如何贴合门洞。 | 支撑“墓门具有封闭结构信息”。 | 不能从砖缝或局部缺损直接推出盗扰、再次封门或人为破坏。 | 技工可用经验口吻提示“看砖层，别只盯一条缝”。 | 可做封门砖层次观察或拆除顺序判断，但不要求真实拆门。 |
| GATE-P1-01 | P1 | 墓门仿木构形式 | 墓门以砖材模拟木构门楼，可见类似梁枋、斗栱或门楼的构件意识。 | 砖面砌出门楼形态，构件像木作，却仍是砖材。 | 墓门表现出砖仿木构特征，可辅助理解地下建筑形象。 | 将门额、门楼构件和铺作样结构归入“砖仿木构”组，并对照知识材料。 | 砖仿木构说明墓门在视觉上模拟地上建筑，但它仍是砖室墓结构。 | 解释墓门为何既是建筑形象又是墓葬入口。 | 不能把它等同真实木构建筑，也不能直接推断墓主等级或官式建筑完全复制。 | 教授可在章节收束时引出“地下宅第”的谨慎解释。 | 可做结构部件归类题，作为 P0 完成后的解释增强。 |
| GATE-P1-02 | P1 | 门前地面关系 | 门前地面与封门砖下缘、门洞边界存在可见交界。 | 封门砖下缘贴近门前地面，交界线还能分出来。 | 门前地面关系可辅助复查封门砖下缘和入口边界。 | 框出门前地面线、封门砖下缘和门洞边界；与 GATE-P0-03 配对。 | 地面关系能帮助玩家把封门砖放回入口结构中，而不是孤立看砖面。 | 辅助支撑“墓门是封闭边界”。 | 不能把地面痕迹写成脚印、停留痕迹或人为事件。 | 技工提醒“下缘也要记，门不是只看上半截”。 | 可作为封门观察任务的补充检查点。 |
| GATE-H-01 | H | 墓门砖缝误读 | 砖缝中可见细砂、灰浆或局部色差，容易被误读为异常标记。 | 砖缝里夹着细砂和灰浆，局部颜色比周围更显眼。 | 砖缝颗粒和色差需复查，暂不能作为异常证据。 | 用放大、标尺或侧光比较周边灰缝；检查它是否形成稳定符号、连续破口或明确结构关系。 | 若颗粒分布不形成稳定符号，也不改变封门结构判断，则降级为工艺/保存细节，并回收到 GATE-P0-03。 | 用于训练“局部细节降级”。 | 不能推成暗号、机关、盗扰、破坏痕迹或人为提示。 | 可让玩家先产生疑问，再由技工/教授要求复查。 | 可做“可疑细节 / 核心证据”分类；降级后解除误导状态。 |
| GATE-R-01 | R | 墓门章节结论 | 由墓门位置、门额正背观察、背面彩画、封门砖组织、门前地面和仿木构复查生成。 | 门额、彩画、封门砖和地面被放到同一张记录里，入口的结构关系清楚了。 | 墓门结论：墓门既是正式入口，也是进入墓室前的第一道结构/装饰证据界面。 | 完成 R 解锁条件后自动生成结论卡。 | 结论只说明墓门的空间和证据功能，不解释墓内事件。 | 接入甬道章节。 | 不能写成破案线索、盗墓现场、机关入口或密室前置。 | 教授收束：“入口本身已经有信息，下一步看甬道如何引导。” | 完成后解锁甬道章节。 |

## 四、H 线索降级规则

| 项目 | 设计 |
| --- | --- |
| 触发 | 玩家点击砖缝、灰缝颗粒、局部色差，或在记录夹中选择“暗号 / 盗扰 / 机关痕迹”等过度解释。 |
| 复查步骤 | 1. 放大观察颗粒分布；2. 与周边灰缝、砖层边界对照；3. 检查是否形成稳定符号、连续破口或明确结构功能；4. 与封门砖整体组织一起判断。 |
| 成功降级反馈 | “这处砖缝值得记录，但它不能单独证明异常。先把它归入封门砖组织和保存状态中。” |
| 失败 / 提示反馈 | “只盯一条缝还不够。回到整面封门砖，看它和门洞、下缘地面的关系。” |
| 降级结果 | `GATE-H-01` 从“待验证”改为“已降级”；不再参与异常结论，只辅助 `GATE-P0-03`。 |

## 五、R 结论解锁条件

| 条件 | 要求 |
| --- | --- |
| 必须获得 | `GATE-E-01`、`GATE-P0-01`、`GATE-P0-02`、`GATE-P0-03` |
| 至少复查 | `GATE-P1-01`、`GATE-P1-02` 至少完成一项，推荐两项都完成 |
| 必须处理 | `GATE-H-01` 必须被降级，或在记录夹中明确标为“待验证，不能用于结论” |
| 生成结论 | `GATE-R-01` 生成后，玩家获得“墓门章节结论卡” |
| 解锁下一章 | 结论卡生成后开放甬道章节 |

## 六、给主线剧情组的直接用法

| 节点 | 可写内容 | 不可写内容 |
| --- | --- | --- |
| 进入墓门 | 技工提醒先确认门的位置、门额和封门砖。 | 不写“破门”“闯入”“机关启动”。 |
| 发现门额背面彩画 | 写成玩家发现第二观察面，产生疑问，随后被要求复查。 | 不写“有人故意藏画”“门额被调包”“彩画暗示真相”。 |
| 观察封门砖 | 写成层次、灰缝、下缘和门洞关系的现场记录。 | 不写“盗墓者从这里进出”“二次封门已坐实”。 |
| 章节收束 | 教授强调“入口本身就是证据界面”。 | 不在墓门阶段解释墓主身份、墓内事件或终章真相。 |

## 八、一页版收束语

墓门章的玩家任务不是“找到秘密入口”，而是学会第一种考古判断：把入口、门额、背面彩画、封门砖和地面关系放在一起复查。它可以有悬疑感，但最终必须收束为谨慎的结构判断。墓门章节完成后，玩家带着“入口也是证据界面”的方法进入甬道，开始阅读通道两壁与顶部图像。
