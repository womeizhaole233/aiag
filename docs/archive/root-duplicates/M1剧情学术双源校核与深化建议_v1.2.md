# M1 剧情 × 学术双源校核与深化建议 v1.2

> 校核依据：
> 剧情源：`D:\考古游戏\文章剧情\白沙尘烟3.docx`、`D:\考古游戏\文章剧情\人物设定.docx`
> 学术源：`D:\考古游戏\白沙宋墓-简体版` 中空间布局、壁画功能、人骨迁葬、随葬品分布、地券、妇人启门、铺作、器物家具等条目。
> 校核对象：当前项目内 M1 已深化文档、全场景线索汇总、全局线索关系矩阵、各场景增强线索表。

## 一、总判断

现有深化方向是对的：应该让玩家通过定位、记录、拓片、测绘、侧光、辨字、叠合和存疑报告，亲手生成结论。

但现有文档中仍混入了一批早期“强悬疑灵感池”的设定。这些设定有游戏张力，但与 `白沙宋墓-简体版` 的学术条目、以及《白沙尘烟3》的最新剧情写法不完全一致。它们不能继续作为 M1 正式学术结论使用。

最重要的修正原则：

> 学术线要回到白沙一号墓真实证据；剧情线可以保留赵怀诚、暗格、反切和绢帛，但必须作为 `F / 剧情推测 / 可推测栏`，不能改写考古 P0。

## 四、可补充深化的地方

### 1. 把“快乐”放在考古动作里

玩家的快乐不应来自“猜中大阴谋”，而应来自：

| 快乐来源 | 对应剧情/学术点 | 可做玩法 |
| --- | --- | --- |
| 看见自己变专业 | 干河道走向、夯层、瓷片标签 | 小测验 + 立即纠错 + 重新记录 |
| 手上动作有反馈 | “永安”拓片、封门砖编号 | 拓片、编号袋、标注坐标 |
| 发现美 | 周淼看东壁女乐，“这个绿怎么调的” | 壁画临摹、颜色卡、乐器识别 |
| 解决真实难题 | 卧丁砖裂缝、架撑选择 | 安全支撑判断小游戏 |
| 知识突然变有用 | 反切注音、《广韵》音系 | 反切拼字小游戏 |
| 读残字像破案但不越界 | 绢帛水渍、偏旁辨认、“知我罪我” | 侧光残字辨认 + 存疑候选 |
| 被 NPC 认可 | 粟柏年“存疑”“还不错” | 结论提交后的审稿反馈 |

### 2. 把赵怀诚线从“真相线”改成“伦理线”

当前剧情最强的地方不是“赵怀诚到底是不是墓主”，而是：

> 他做了一件可能违法但救人的事，并把评判权交给后人。

这比机关、盗墓、诅咒更高级，也更适合学术克制的游戏。

建议终章让玩家提交：

| 栏目 | 玩家可写 |
| --- | --- |
| 学术可判断 | M1 年代、赵姓称谓、空间结构、前堂后寝、迁葬证据、地券制度。 |
| 剧情可推测 | 赵怀诚可能借墓中隐文和绢帛留下自述。 |
| 伦理可讨论 | 私调官粮救灾该如何评价。 |
| 仍存疑 | 赵怀诚与赵大翁是否完全同一、暗格是否真实存在于考古材料、绢帛内容能否外证。 |

## 五、现有深化结论修订版

### 学术最终结论

M1 是一座北宋元符二年左右的赵姓民间精英砖室墓。它具有墓道、墓门、甬道、前室、过道、后室的连续空间序列，墓门和墓室以砖仿木构模拟地上建筑，前室通过乐舞、宴饮、贡纳、侍卫和器物图像形成礼仪展示，过道题记提供年代与称谓锚点，后室以砖床、人骨、铁钉、地券、镇墓钱和随葬器物构成安葬核心，并以妇人启门、假门和日常器物图像营造地下寝居。

### 剧情阶段结论

在不冒充考古定论的前提下，赵怀诚剧情线可以这样收束：

> “赵怀诚”把姓名、罪责和辩白拆散藏在墓中。玩家通过屏风隐字、暗格铜筒、反切素帛、绢帛残字和时间线复查，把这份自述重新拼合出来。但它最终只能是墓中自述，不能直接等同官方史实。

### 玩家体验结论

玩家最后获得的不是“破案成功”，而是：

1. 会看地形、夯层、封门和空间序列。
2. 会读壁画功能，而不是只看漂亮图像。
3. 会把题记、地券、人骨、铁钉、器物放在一起判断。
4. 会把剧情推测和学术结论分开。
5. 会在报告里写下“仍存疑”。

## 六、下一步我建议怎么做

### 第一步：先清洗错误线索

目标：把所有正式文档中的硬错误先改掉。

优先修：

1. `M1/16_出土器物与人骨` 所有“四具人骨、八枚铁钉、地券十二行、乡级缺失、涂改”。

目标：只保留能被剧情源和学术源共同支撑，或能被明确分栏的内容。

正式主链：

### 提示词 4：接入赵怀诚剧情线

```text
请在不破坏白沙宋墓学术结论的前提下，接入赵怀诚剧情线。要求：
```

原因：赵怀诚线是情感发动机，但必须放在正确栏位里，才能既好玩又不伤学术严谨。

## 八、一句话结论

现有深化最大的方向没错，但必须先把“强悬疑灵感池”从“正式学术证据链”里清出来。真正能让玩家快乐的，不是把墓写成阴谋现场，而是让玩家在一次次观察、动手、复查、被 NPC 轻轻纠正之后，发现自己真的变得更会看一座墓。
