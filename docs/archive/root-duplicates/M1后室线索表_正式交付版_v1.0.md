## 后室章节小结论目标

后室是 M1 的安葬核心区，同时通过假门、妇人启门和日常器物图像，被塑造成具有地下寝居意味的空间。玩家需要把砖床、人骨、铁钉、地券、券盖和壁画器物放回同一空间中复查，而不是依靠单个异常得出结论。
