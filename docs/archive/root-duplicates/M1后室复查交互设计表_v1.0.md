# M1 后室复查交互设计表 v1.0

> 基于 `M1后室图片热点线索流程映射表_v1.0.md`。
>
> 本表只做交互设计，不改代码。后室复查分为两个交互：`遗存定位` 和 `器物归类`。目标不是制造新事实，而是让玩家把已发现的线索放回空间关系中，完成谨慎判断。

## 一、复查交互总览

| 交互 | 主要材料 | 目的 | 产出 |
|---|---|---|---|
| 遗存定位 | 出土物分布图、后室平剖面、砖床、人骨、铁钉、地券、白瓷碗 | 建立“砖床 -> 人骨/铁钉/地券/器物”的空间关系 | 完成 P0 安葬核心链，降级 HS-H-01/HS-H-02 |
| 器物归类 | 北壁假门、南壁高几、西南壁镜台/盆架/杌、东北壁灯具、西北壁剪刀熨斗、东南壁/顶部/铺作 | 建立“后室多壁面 -> 日常化陈设 -> 地下寝居想象”的图像关系 | 完成 P1 生活化图像链，降级 HS-H-03 |

## 二、复查交互一：遗存定位

### 1. 触发条件

| 条件 | 要求 | 未满足时提示 |
|---|---|---|
| 进入后室 | 已从过道进入后室，并获得 HS-E-01 | “先确认你已经进入后室，再查看出土记录。” |
| 发现砖床 | 已点击 `rear_lower_brick_bed` 或 `rear_north_lower_overview` | “先找到北壁下方的砖床区域。” |
| 打开分布图 | 已解锁 `rear_excavation_distribution` | “出土物位置需要在分布图中复查。” |
| 可选前置 | 已点击人骨、铁钉、地券任意一项 | “可以先从砖床开始定位，再逐步补入遗存。” |

### 2. 玩家操作流程

| 步骤 | 玩家操作 | 成功条件 | 成功反馈 | 失败/提示反馈 | 更新线索记录 |
|---|---|---|---|---|---|
| B1 | 在分布图上框出砖床范围 | 框选区域落在后室砖床对应范围内 | “砖床范围已标出。其他遗存可以放回这个基准中。” | “先找低平、成片的砖床区域，不要从单件器物开始。” | HS-P0-01 标记为 `已定位` |
| B2 | 把人骨标记拖到对应位置 | 人骨标记位于砖床核心区域附近 | “人骨位置已记录。它需要和砖床、铁钉一起复查。” | “人骨不是整间后室的范围，注意它在砖床内的位置。” | HS-P0-02 标记为 `已定位` |
| B3 | 把铁钉/葬具范围标到图上 | 铁钉范围与人骨相邻或围合，但不完全等同于人骨边界 | “铁钉范围已记录。它更适合作为葬具复原线索。” | “铁钉不是人骨本身，也不能直接当成破坏证据。” | HS-P0-03 标记为 `已复查`; HS-H-02 标记为 `待降级` |
| B4 | 把地券/券盖标到对应位置 | 地券标记回到后室出土位置，并能关联地券照片/并盖图 | “地券位置已记录。它和砖床、人骨属于不同类型的证据。” | “地券不是壁画，也不是普通砖块。先回到出土位置。” | HS-P0-04 标记为 `已定位` |
| B5 | 把白瓷碗标到随葬器物组 | 白瓷碗被归入随葬器物，而不是主线异常或暗号 | “白瓷碗已归入随葬器物。它目前不能单独承担主线判断。” | “这件器物值得记录，但还不能把它当作暗号或身份标记。” | HS-H-01 标记为 `已降级`; HS-P1-05 标记为 `已补充` |
| B6 | 点击“完成遗存定位” | B1-B5 中 B1-B4 必须完成，B5 可完成降级 | “遗存定位完成：砖床、人骨、铁钉、地券已建立空间关系。” | “还缺少关键遗存定位：请检查砖床、人骨、铁钉、地券。” | 生成状态 `rear_relic_position_completed = true` |

### 3. H 线索降级规则

| H 线索 | 误读方向 | 降级动作 | 降级后归属 |
|---|---|---|---|
| HS-H-01 白瓷碗位置 | 暗号、外来者证据、特殊仪式核心 | 在分布图中归入“随葬器物/待验证器物”，并确认无明确文字或刻款 | 并入 HS-P1-05，作为随葬器物分布补充 |
| HS-H-02 铁钉数量与位置差异 | 盗扰、二次开棺、尸骨移动 | 将铁钉范围作为葬具复原线索，与人骨范围分开标注 | 回收至 HS-P0-03，作为葬具痕迹复查结果 |

### 4. 遗存定位完成状态

| 状态 | 判断 |
|---|---|
| 未完成 | 只点击了单张图片，没有在分布图中标注关系 |
| 部分完成 | 已标注砖床和人骨，但缺少铁钉或地券 |
| 完成 | 砖床、人骨、铁钉、地券均已定位，且至少一条 H 线索完成降级 |

## 三、复查交互二：器物归类

### 1. 触发条件

| 条件 | 要求 | 未满足时提示 |
|---|---|---|
| 多壁面入口开启 | 已获得 HS-E-02 或打开后室平、剖面图 | “后室不只看北壁，先打开后室多壁面复查。” |
| 妇人启门已观察 | 已获得 HS-P0-05 | “先确认北壁假门和妇人启门，再整理侧壁器物。” |
| 至少发现一件器物 | 镜台、盆架、杌、高几、灯具、剪刀熨斗任意一件 | “先在后室各壁面寻找可归类的器物。” |

### 2. 归类板分区

| 分区 | 可放入对象 | 支撑线索 |
|---|---|---|
| 梳妆/坐具组合 | 镜台、曲足盆架、杌 | HS-P1-01 |
| 承置/陈设 | 高几 | HS-P1-02 |
| 照明/日用器物 | 灯具、剪刀、熨斗 | HS-P1-03; HS-P1-04 |
| 空间结构/壁面 | 后室入口背面、东南壁、东北壁、西北壁、顶部、铺作 | HS-E-02 |
| 待降级误读 | 假门后的空间暗示 | HS-H-03 |

### 3. 玩家操作流程

| 步骤 | 玩家操作 | 成功条件 | 成功反馈 | 失败/提示反馈 | 更新线索记录 |
|---|---|---|---|---|---|
| C1 | 把镜台、曲足盆架、杌放入同一组 | 三件器物均归入“梳妆/坐具组合” | “三件器物形成一组室内陈设。” | “不要只看单件器物，先找同一壁面的组合关系。” | HS-P1-01 标记为 `已归类` |
| C2 | 把高几放入“承置/陈设” | 高几没有被单独判为祭桌或供桌 | “高几已归入陈设线索。” | “单件高几还不能直接说明祭祀用途。” | HS-P1-02 标记为 `已归类` |
| C3 | 把灯具放入“照明/日用器物” | 灯具作为图像中的照明器物处理 | “灯具已归入生活器物组。” | “图像中有灯具，不等于墓室曾实际点灯。” | HS-P1-03 标记为 `已归类` |
| C4 | 把剪刀、熨斗放入“照明/日用器物”或“日用器物” | 剪刀、熨斗作为日用器物处理，不推出身份 | “剪刀、熨斗已归入日用器物组。” | “这些器物不能直接证明墓主性别或职业。” | HS-P1-04 标记为 `已归类` |
| C5 | 把东南壁、东北壁、西北壁、南壁入口、顶部、铺作放入“空间结构/壁面” | 至少完成北壁、南壁、西南壁、东北/西北壁、顶部五组视角 | “多壁面复查完成。后室不是单一图像空间。” | “后室结论不能只看北壁，还需要侧壁和顶部。” | HS-E-02 标记为 `多壁面已复查` |
| C6 | 把假门空间暗示放入“图像空间/待降级误读” | 玩家选择“图像中的空间暗示”，不选择“真实通道/机关” | “假门保留为空间暗示，不作为真实通道证据。” | “当前证据不足以支持密室或机关判断。” | HS-H-03 标记为 `已降级`; HS-P0-05 更新为 `已复查` |
| C7 | 点击“完成器物归类” | HS-P1-01 必须完成，HS-P1-02/03/04 至少完成两条，HS-E-02 完成 | “器物归类完成：后室壁画呈现出日常陈设和多壁面空间关系。” | “还缺少关键器物组合或多壁面复查。” | 生成状态 `rear_object_group_completed = true` |

### 4. H 线索降级规则

| H 线索 | 误读方向 | 降级动作 | 降级后归属 |
|---|---|---|---|
| HS-H-03 假门后的空间暗示 | 密室、机关、真实通道 | 用假门测绘图、北壁总览和妇人特写复查，选择“图像空间暗示” | 回收至 HS-P0-05，作为妇人启门和假门图像组合的谨慎解释 |

### 5. 器物归类完成状态

| 状态 | 判断 |
|---|---|
| 未完成 | 只点击单件器物，未完成归类 |
| 部分完成 | 已完成镜台/盆架/杌组合，但缺少其他器物或多壁面复查 |
| 完成 | HS-P1-01 完成，HS-P1-02/03/04 至少两条完成，HS-E-02 完成，HS-H-03 已降级 |

## 四、HS-R-01 后室章节结论解锁

### 1. 解锁条件

| 条件组 | 必须满足 | 对应状态 |
|---|---|---|
| P0 安葬核心链 | HS-P0-01、02、03、04、05 全部进入记录夹 | `rear_core_p0_completed = true` |
| 遗存定位 | 砖床、人骨、铁钉、地券完成分布图定位 | `rear_relic_position_completed = true` |
| P1 器物解释链 | HS-P1-01 必须完成，HS-P1-02/03/04 至少完成两条 | `rear_object_group_completed = true` |
| 环境/多壁面 | HS-E-01、HS-E-02 均完成 | `rear_space_review_completed = true` |
| H 降级 | HS-H-01/02/03 至少降级一条，推荐 HS-H-03；若三条都降级，可给“谨慎判断”强化反馈 | `rear_h_downgraded_count >= 1` |

### 2. 解锁流程

| 步骤 | 玩家动作 | 系统判断 | 结果 |
|---|---|---|---|
| R1 | 打开记录夹中的“后室汇总” | 检查 P0、遗存定位、器物归类、E、H 降级状态 | 若缺项，显示缺项提示 |
| R2 | 点击“生成后室小结论” | 所有条件满足 | 生成 HS-R-01 |
| R3 | 查看 HS-R-01 | 结论卡进入记录夹 | 解锁终章大结论所需的后室结论节点 |

### 3. 缺项提示文案

| 缺项 | 提示 |
|---|---|
| 缺 P0 | “后室核心证据还不完整：请复查砖床、人骨、铁钉、地券和妇人启门。” |
| 缺遗存定位 | “出土记录还没有放回分布图：请完成遗存定位。” |
| 缺器物归类 | “后室器物还没有形成组合：请完成器物归类。” |
| 缺 E02 | “后室不能只看单面墙：请完成多壁面复查。” |
| 缺 H 降级 | “还有误读没有排除：请至少降级一条待验证线索。” |

### 4. HS-R-01 生成文案

| 位置 | 文案 |
|---|---|
| 弹窗 | “你把砖床、人骨、铁钉、地券和壁画器物放回同一张后室平面。分开的发现开始互相对上位置。” |
| 记录夹 | “后室章节结论：后室既是 M1 的安葬核心区，也是被图像塑造成地下寝居的空间。” |
| 复查反馈 | “P0 线索确认安葬核心；P1 线索补充生活化图像；E 线索确认空间收束；H 线索已降级为待验证或误读排除。” |

## 五、设计边界

| 禁止做法 | 原因 |
|---|---|
| 遗存定位后直接给出盗墓、谋杀或二次葬结论 | 现有复查只能支持空间关系和葬具复原，不能支持事件真相 |
| 把白瓷碗设计成暗号核心 | 目前缺少文字、刻款或异常层位支撑 |
| 把剪刀、熨斗、镜台直接推出墓主性别 | 器物图像只能支持生活化陈设，不足以直接证明身份 |
| 把假门设计成真实密室入口 | 妇人启门和假门应作为图像母题与空间暗示处理 |
| 强迫所有顶部/铺作/东南壁图片都生成独立线索 | 容易稀释主线，建议纳入 HS-E-02 多壁面复查 |
