# M1 后室线索精修版 v1.0

> 基于《M1各章节线索母表_v1.0.md》和已完成的墓门、甬道、前室、过道精修版标准整理。
> 本版只打磨线索本身，暂不做图片、热点、坐标或代码映射。

## 一、章节定位

| 项目 | 内容 |
| --- | --- |
| 核心学术问题 | 后室如何通过砖床、人骨、铁钉与葬具痕迹、地券、妇人启门、假门和日常器物图像，共同构成安葬核心与地下寝居空间？ |
| 章节小结论 | 后室既是 M1 的安葬核心区，也是被图像塑造成地下寝居的空间。这个判断必须来自空间位置、实体遗存、葬具痕迹、葬仪文书、假门图像和多壁面器物图像的组合。 |
| 主线剧情可用方向 | 后室可以作为章节高潮，但剧情必须克制。悬疑来自证据集中和复查压力，不来自谋杀、盗墓、机关密室或超自然真相。 |
| 小游戏可用复查方向 | 后室遗存定位、葬具范围复查、地券文书辨识、器物归类、假门误读降级、章节结论卡生成。 |
| 证据边界 | 本章只能支持“安葬核心区 + 地下寝居图像表达”。不能把单件器物、人骨位置、铁钉数量、白瓷碗、假门或妇人启门写成身份真相、死因、盗墓、二次开棺、密室、机关或超自然证据。 |

## 七、给主线剧情组的直接用法

| 节点 | 可写内容 | 不可写内容 |

| 观察砖床与人骨 | 写成砖床、人骨、铁钉需要叠合定位。 | 不写死因、谋杀、盗墓、迁葬已坐实或身份揭示。 |
| 观察地券 | 写成朱书砖质文书和券盖进入葬仪制度判断。 | 不写诅咒、涂改阴谋、身份抹除或神秘文书。 |

## 九、一页版收束语

后室章的玩家任务不是“发现最终真相”，而是学会第五种考古判断：把实体遗存、葬仪文书、图像母题和日常器物放回同一个空间中复查。砖床、人骨和铁钉确认安葬核心；地券和随葬器物补充葬仪信息；妇人启门与假门制造图像中的空间延续；镜台、高几、灯具、剪刀熨斗等器物让后室呈现地下寝居感。只有这些证据合读，后室才能被谨慎理解为 M1 的安葬核心区与图像化地下居所。
