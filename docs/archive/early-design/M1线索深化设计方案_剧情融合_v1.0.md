~~~~# M1 线索深化设计方案：学术材料 ×《白沙尘烟3》剧情融合 v1.0

> 依据：`D:\考古游戏\白沙宋墓-简体版` 学术材料、`D:\考古游戏\文章剧情\白沙尘烟3.docx` 剧情文本、`D:\考古游戏\文章剧情\人物设定.docx` 人物设定，以及已完成的 M1 各章线索精修版。
> 本文用于指导下一轮线索深化：在不破坏学术谨慎性的前提下，让线索更有剧情牵引、复查乐趣和玩家参与感。

## 一、当前线索的问题

现在的 M1 线索体系已经完成了“学术稳态”：

- 每章都有 E/P0/P1/H/R 层级。
- 点击弹窗以自然观察为主。
- 章节结论不做过度推断。
- H 线索用于降级，防止玩家把单点异常推成盗墓、谋杀、机关密室或超自然解释。

但如果直接进入游戏，会有三个问题：

| 问题 | 表现 | 需要补强 |
| --- | --- | --- |
| 线索偏“说明” | 玩家知道它重要，但不一定想继续追 | 给线索加入“悬念钩子”和“后续回收” |
| 学术判断偏静态 | 结论正确，但玩家参与感不足 | 把判断过程做成可操作的复查任务 |
| 剧情与线索未咬合 | 赵怀诚、暗格、反切、绢帛、银扣等剧情内容还没有进入线索系统 | 建立“学术线索 + 剧情线索 + 玩法动作”的三层结构 |

## 二、深化原则

### 1. 不替换原有学术线索，而是在其上叠加剧情层

原有 P0/P1/E/R 负责学术结论，不能被剧情内容随意改写。

新增或强化的剧情内容可以作为“剧情增强字段”，但不要直接变成学术 P0，除非它确实有考古材料或文本证据支撑。

推荐字段：

| 字段 | 用途 |
| --- | --- |
| 学术证据功能 | 这条线索在考古判断中支撑什么 |
| 剧情悬念功能 | 它让玩家产生什么疑问 |
| 玩家复查玩法 | 玩家要做什么动作才能确认或排除 |
| NPC 反馈角色 | 谁最适合解释或反驳 |
| 回收节点 | 这条线索会在哪一章或终章被重新理解 |

### 2. 结论采用“双轨制”

每章可以同时拥有两种结论：

| 结论类型 | 作用 | 示例 |
| --- | --- | --- |
| 学术小结论 | 给主线考古判断使用 | “后室既是安葬核心区，也是地下寝居图像空间。” |
| 剧情阶段结论 | 给玩家悬念推进使用 | “赵怀诚似乎主动把某件事藏进墓中，等待后人读出。” |

注意：剧情阶段结论不能覆盖学术结论。它可以推动玩家继续查，但不能冒充考古定论。

### 3. H 线索不只是“误导”，也可以是“好奇心入口”

H 线索现在的功能是降级，这是对的。但在玩法上，它可以先吸引玩家：

- 让玩家先产生解释冲动。
- 通过复查发现证据权重不够。
- 降级后回收到 P0/P1 证据链。

这样 H 线索不会变成“废线索”，而会变成训练玩家方法意识的小游戏。

### 4. 让 NPC 成为不同类型证据的“解释器”

剧情人物本身可以绑定线索玩法：

| 角色 | 适合处理的线索 | 玩法功能 |
| --- | --- | --- |
| 粟柏年 | 地层、文字判断、结论边界 | 纠偏、追问、最终收束 |
| 周淼 | 壁画、纹饰、颜料、反切字形 | 细节发现、图像复原、文字辅助 |
| 陈怀远 | 封门、支撑、砖构、平剖面 | 结构判断、安全复查、测绘任务 |
| 苏池 | 人骨、年龄、性别、迁葬可能 | 遗存复查、证据边界 |
| 赵老倔 | 土层经验、墓外传闻、现场直觉 | 氛围钩子、经验判断、传闻降级 |
| 林砚秋 | 玩家视角、笔记、银扣、选择 | 连接学术线与个人线 |

## 三、M1 可以深化成三条主线

### A. 学术证据线

这条线保持现有体系：

墓外定位 -> 墓门结构 -> 甬道引导 -> 前室展示 -> 过道时间锚点 -> 后室安葬核心 -> 终章证据系统。

玩家最终获得谨慎学术结论：

> M1 是由空间、图像、文书、遗物和遗骸共同构成的北宋砖室墓证据系统。

### B. 赵怀诚剧情线

这是《白沙尘烟3》里最强的故事线，应成为 M1 的剧情深层线：

| 阶段 | 剧情线索 | 玩家疑问 |
| --- | --- | --- |
| 墓门 | “永安”刻字 | 这是祝福、地名、堂号，还是某种自我命名？ |
| 前室/过道 | “元符二年赵大翁布” | 赵大翁是谁？只是墓主称谓，还是有人刻意留下的身份锚点？ |
| 后室北壁 | 屏风/山水中隐约文字 | 墓主是否在图像里藏了话？ |
| 暗格 | 铜筒、素帛、反切 | 为什么要把名字拆开藏起来？ |
| 绢帛 | 崇宁四年水灾、私调官粮、知我罪我 | 赵怀诚希望后人如何评价他？ |
| 后室清理 | 1099 年建墓与 1106 年绢帛时间差 | M1 最初是否并非为赵怀诚最终安葬而建？ |

剧情阶段结论：

> 赵怀诚把自己的名字、罪责和辩白拆散藏入墓中，让后来进入墓的人通过复查、辨字和推理重新拼合。

### C. 玩家成长线

林砚秋不是单纯“点线索”的玩家壳，她的成长可以通过玩法体现：

| 阶段 | 玩家能力 | 对应玩法 |
| --- | --- | --- |
| 墓外 | 学会看地形 | 干河道走向选择、地图定位 |
| 墓道/墓门 | 学会先记录再解释 | 夯层观察、瓷片标签、封门拆除顺序 |
| 甬道 | 学会三面合读 | 顶部、东壁、西壁同时入记录夹 |
| 前室 | 学会图像功能判断 | 乐舞、宴饮、人物、器物归类 |
| 过道 | 学会把文字当锚点 | 题记辨读，但不能一证独大 |
| 后室 | 学会多证据合读 | 人骨、铁钉、地券、暗格、绢帛、壁画互证 |
| 终章 | 学会写谨慎结论 | 把“知道什么”和“不能确定什么”分开提交 |

## 四、各章节深化建议

| 章节 | 现有学术结论 | 建议新增/强化的剧情钩子 | 推荐玩法 | 结论边界 |
| --- | --- | --- | --- | --- |
| 墓外/环境 | M1 应作为墓群环境中的完整空间序列进入 | 外祖父银扣、干河道走向、赵老倔的地方经验 | 地形观察选择、银扣记忆卡、传闻/证据分类 | 银扣只作个人线伏笔，不在 M1 内强行解释 |
| 墓道/墓门 | 墓门是入口和第一道结构/装饰证据界面 | “永安”刻字、封门三层拆除、卧丁砖裂缝 | 拓片、拆砖顺序、安全支撑判断 | “永安”先存疑，不能直接定为堂号或身份真相 |
| 甬道 | 甬道承担过渡与引导 | 版门断砌造、门钉、叠涩磨损、抬棺经过痕迹 | 三面合读、构件标注、磨损来源复查 | 磨损只能提示使用/通过可能，不能直接还原下葬现场 |
| 前室 | 前室承担礼仪展示、人物陈设和结构转折 | 开芳宴的“宴/祭”双重意味、墓主夫妇浮雕与壁画层次 | 乐舞器物归类、主次关系判断、侧光找隐字 | 不把宴饮图直接写成真实宴会或唯一身份揭示 |
| 过道 | 过道集中出现时间信息和空间转换信息 | “元符二年赵大翁布”作为身份锚点，连接后室地券和绢帛 | 题记辨读、称谓解释、时间轴排序 | 题记可定时间和称谓，不能解释全部故事 |
| 后室 | 后室是安葬核心区和地下寝居图像空间 | 假门凹槽、暗格铜筒、反切素帛、赵怀诚绢帛、迁葬存疑 | 假门复查、反切拼字、残字辨认、人骨葬具叠合 | 可形成剧情判断，但迁葬原因、初葬对象仍需存疑 |
| 终章 | M1 的意义来自多类证据共同指向 | 玩家整理“赵怀诚是谁、他留下了什么、哪些仍不能确定” | 证据墙、时间线、结论报告、存疑栏 | 不给破案式唯一真相；保留“知我罪我”的伦理余味 |

## 五、重点线索深化样例

### 1. “永安”刻字

| 字段 | 设计 |
| --- | --- |
| 学术证据功能 | 墓门封门砖或门前砖面上的文字/刻痕，需要拓片和位置记录。 |
| 剧情悬念功能 | 让玩家第一次遇到“墓中有人主动留下文字”的感觉。 |
| 玩家复查玩法 | 拓片小游戏：先清理、覆纸、拓墨，再辨字。 |
| 点击弹窗 | 砖面上刻着两个工整的字，刻痕里还压着干硬的泥。 |
| 记录夹文案 | “永安”二字已拓录，含义暂不能判断。需等待后续文字证据。 |
| NPC 反馈 | 粟柏年：“存疑。先别写进结论。” |
| 回收节点 | 过道题记、后室地券、暗格绢帛出现后，玩家再回看“永安”的性质。 |
| 边界 | 不能直接解释成堂号、暗号、机关口令或墓主身份。 |

### 2. 屏风隐字

| 字段 | 设计 |
| --- | --- |
| 学术证据功能 | 壁画局部墨线与颜料层观察，属于待复查文字痕迹。 |
| 剧情悬念功能 | 从“看图像”转向“图像里藏着文字”。 |
| 玩家复查玩法 | 侧光移动：玩家调整手电角度，墨线在裂隙中浮现。 |
| 点击弹窗 | 侧光掠过屏风山水，裂纹里有几道极细的墨线。 |
| 记录夹文案 | 屏风中发现疑似隐写文字，需逐字辨认并与后续文书互证。 |
| NPC 反馈 | 周淼负责提示线条，粟柏年负责提醒不要急于定性。 |
| 回收节点 | 与暗格绢帛中的赵怀诚自述互相印证。 |
| 边界 | 不能只凭隐字直接推出完整历史事件；必须等待绢帛和地券。 |

### 3. 反切素帛

| 字段 | 设计 |
| --- | --- |
| 学术证据功能 | 宋代文字材料和音韵知识卡，不直接属于墓葬结构证据，但属于剧情文书核心。 |
| 剧情悬念功能 | 赵怀诚把名字拆成谜题，玩家必须亲手拼回姓名。 |
| 玩家复查玩法 | 反切拼字：上字取声母，下字取韵母和声调，结合《广韵》知识卡。 |
| 点击弹窗 | 素帛上只有两行小字，字数很少，却不像普通题记。 |
| 记录夹文案 | 素帛使用反切方式隐藏姓名信息；“治小”可拼出“赵”，“是征”指向“诚”。 |
| NPC 反馈 | 周淼先发现“赵”，粟柏年补充反切规则。 |
| 回收节点 | 与绢帛首句“怀诚谨记”合成“赵怀诚”。 |
| 边界 | 不能把反切解谜做成纯机关密码；它应是文字学方法和剧情身份确认。 |

### 4. 绢帛残字

| 字段 | 设计 |
| --- | --- |
| 学术证据功能 | 暗格出土文书，记录需标明材质、位置、封存状态和残损。 |
| 剧情悬念功能 | 让玩家从“赵怀诚是谁”进入“赵怀诚做了什么”。 |
| 玩家复查玩法 | 残字辨认：侧光观察偏旁、根据上下文选择候选字。 |
| 点击弹窗 | 绢帛下端被水渍浸过，关键几处字迹只剩偏旁。 |
| 记录夹文案 | 绢帛记录崇宁四年水灾、私调官粮与赵怀诚自述，残损处需谨慎补读。 |
| NPC 反馈 | 粟柏年要求逐字辨认；周淼补充典故“知我罪我”。 |
| 回收节点 | 形成 M1 剧情阶段结论：墓内隐藏的是一份等待后人评判的自述。 |
| 边界 | 不能把绢帛变成“官方史实”；它是墓中自述文书，需要与其他材料互证。 |

### 5. 人骨、铁钉与迁葬存疑

| 字段 | 设计 |
| --- | --- |
| 学术证据功能 | 后室安葬核心证据，支持葬具范围和迁葬可能。 |
| 剧情悬念功能 | 1099 年建墓与 1106 年绢帛时间差，让玩家意识到安葬过程可能有阶段。 |
| 玩家复查玩法 | 人骨范围、铁钉轮廓、棺木尺寸、时间线叠合。 |
| 点击弹窗 | 铁钉排出一圈棺木轮廓，但骨骼并没有完整躺在里面。 |
| 记录夹文案 | 后室遗骸与葬具痕迹支持迁葬可能，但迁葬原因和初葬位置仍需存疑。 |
| NPC 反馈 | 苏池给出体质人类学判断，粟柏年在玩家笔记旁写“存疑”。 |
| 回收节点 | 终章时间线：元符二年建墓、崇宁四年事件、崇宁五年绢帛。 |
| 边界 | 不能推出谋杀、盗墓、仓促弃尸或身份真相。 |

| 玩法模块 | 使用章节 | 玩家操作 | 价值 |

| 三面合读 | 甬道 | 同时记录顶部、东壁、西壁 | 防止只点单个壁画 |

| 存疑报告 | 终章 | 把“可判断”和“不可判断”分栏提交 | 形成学术克制的最终体验 |

## 八、最终体验目标

深化后的 M1 不应只是“点线索 -> 得结论”，而应是：

1. 玩家先学会田野方法：看地形、记位置、做拓片、画图。
2. 玩家再学会考古判断：结构、图像、文书、遗存不能单独解释。
3. 玩家逐步进入剧情核心：赵怀诚把名字和自述拆散，藏在墓中。
4. 玩家亲手完成三个拼合：
   - 空间拼合：墓外到后室。
   - 证据拼合：结构、图像、文书、遗存。
   - 姓名拼合：赵 + 怀 + 诚。
5. 玩家最终提交的不是“真相大白”，而是一份带有存疑栏的判断：
   - 学术上：M1 是多类证据共同构成的北宋砖室墓系统。
   - 剧情上：赵怀诚留下了一份等待后人评判的自述。
   - 方法上：有些事可以判断，有些事只能谨慎存疑。

一句话目标：

> 让玩家不是“看见结论”，而是亲手把结论从尘土、墙面、文字和遗骨中拼出来。

## 九、本轮继续深化落地说明

本轮已将本文的原则进一步落到一份可交付表：

`M1各章节线索母表_剧情深化版_v1.1.md`

这份 v1.1 不替代 `M1各章节线索母表_v1.0.md`，而是作为剧情、玩法和代码接入的增强层。它把现有线索逐条补齐以下字段：

| 新增字段 | 解决的问题 |
| --- | --- |
| 剧情悬念功能 | 避免线索只是说明文字，让玩家知道为什么要继续查。 |
| 角色触发/反馈 | 明确谁发现、谁解释、谁质疑、谁收束，方便 NPC 对话组接入。 |
| 玩家操作玩法 | 把“点击获得”改成定位、拓片、侧光、辨读、归类、叠合、降级等考古动作。 |
| 回收节点 | 让前文线索在后室或终章被重新理解，形成剧情牵引。 |
| 剧情边界 | 明确每条线索不能推成盗墓、谋杀、机关、密室、身份真相或超自然解释。 |
| 建议数据处理 | 给程序侧区分 `observation/check/hypothesis/relation/F`，避免点击即结论。 |

### 1. 本轮最重要的结构变化

原有线索母表的主轴是：

> 线索事实 -> 复查动作 -> 章节小结论。

v1.1 增强后的主轴是：

> 线索事实 -> 剧情疑问 -> 玩家复查 -> NPC 质疑/收束 -> 章节结论或存疑栏。

这个变化的目的不是让 M1 更“悬疑化”，而是让悬疑感必须经过考古方法过滤。玩家可以好奇，可以误判，也可以提出假说，但每一次假说都要被复查动作和证据边界校准。

### 2. 新增剧情线索的隔离规则

赵怀诚、银扣、永安刻字、屏风隐字、暗格、反切素帛、绢帛自述和时间差等内容，已经在 v1.1 中统一标出接入边界：

| 剧情内容 | 推荐处理 |
| --- | --- |
| 银扣 | `F`，林砚秋个人线伏笔，不进入 M1 学术结论。 |
| “永安”刻字 | `F / 待核`，若无真实材料来源，只作剧情文字；不得直接解释为堂号、暗号或身份。 |
| 屏风隐字 | `F / 待核`，只作“疑似文字痕迹”和侧光玩法入口。 |
| “赵大翁”身份锚点 | 题记本体仍按 P0 处理；“赵大翁是谁”的追问进入剧情增强字段。 |
| 暗格 | `F / 待核`，必须和假门学术判断分开，不得把假门改写成密室机关。 |
| 反切素帛 | `F`，用于文字学解谜和姓名拼合，不作为墓葬结构证据。 |
| 绢帛自述 | `F`，作为剧情文书，需和地券、题记、人骨时间线分栏互证。 |
| 1099 / 1106 时间差 | 剧情增强字段，可引出阶段性安葬疑问，但迁葬原因必须存疑。 |

### 5. 本轮深化后的判断边界

v1.1 允许 M1 具有更强剧情牵引，但最终仍需坚持：

1. 学术上，M1 的大结论来自空间、图像、文书、遗物和遗骸合读。
2. 剧情上，赵怀诚线只能作为玩家追问和阶段性判断，不能覆盖学术结论。
3. 玩法上，所有强悬念必须经过复查或降级，不允许单点异常直接变成真相。
4. 终章上，玩家必须把“能判断的”和“只能存疑的”分开提交。
