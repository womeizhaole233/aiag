# M1 剧情章节节点与 NPC 任务梳理表

> 本表根据《白沙尘烟剧情3》、现有 PRD 与团队分工文档整理。用途是帮助固定剧情 NPC 负责人、自由对话 NPC 负责人、线索负责人和小游戏负责人对齐：每章发生什么、在哪个场景发生、有哪些 NPC、NPC 要做什么、对应哪些小游戏。

## 1. 总体章节结构

当前剧情稿适合整理成 8 个章节节点：

| 章节 | 剧情稿对应段落 | 游戏场景 | 当前定位 |
|---|---|---|---|
| CH01 抵达白沙 | 楔子 | 墓外 / 白沙镇 / 工地入口 | 建立玩家身份、考古队关系和空间背景 |
| CH02 墓道初清 | 墓道 | 墓道 / 墓门前 | 学习地层、记录、清理与封门前观察 |
| CH03 封门与墓门 | 墓道后段 + 封门砖 | 墓门 / 封门砖 | 判断封门结构、拆除顺序和安全支撑 |
| CH04 甬道入室 | 甬道 | 甬道顶部 / 东西壁 / 版门 | 进入墓室，观察通道结构、版门、顶部彩画 |
| CH05 前室礼仪空间 | 前室 | 前室入口总览 / 东西南北壁 / 顶部 | 观察礼仪图像、宴饮、女乐、藻井和题记 |
| CH06 后室北壁与剧情分歧 | 后室 | 后室入口总览 / 后室北壁 | 观察妇人启门，进入后室核心图像 |
| CH07 暗格与剧情谜题 | 暗格 | 后室西北角 / 暗格 / 铜筒 | 剧情创作线，反切、绢帛、屏风藏字 |
| CH08 后室清理与封存 | 后室清理与封存 | 后室砖床 / 地券 / 出土物分布 | 人骨、铁钉、地券、随葬品和章节收束 |

说明：

1. CH01-CH06、CH08 可直接转化为固定剧情节点与游戏章节。
2. CH07 暗格、铜筒、绢帛、屏风藏字属于剧情创作线，应标为 F 类剧情线索，不作为真实考古证据。
3. 当前 MVP 小游戏只保留：挖土消消乐、题记辨读、后室遗物定位。

## 3. NPC 角色任务表

| NPC | 身份 | 固定剧情任务 | 自由对话可回答范围 | 不应承担 |
|---|---|---|---|---|
| 粟柏年 | 考古领队 / 教授 | 提出观察问题、解释考古方法、反馈选择、章节收束 | 当前目标、缺失线索、术语解释、记录夹复查方向 | 不直接告诉所有答案，不提前剧透后室或暗格结论 |
| 周淼 | 壁画与纹饰测绘队员 | 接引玩家、观察壁画、提示颜色、纹样、乐器与人物细节 | 壁画、纹饰、颜色、测绘、东壁女乐 | 不解释人骨、结构力学、最终结论 |
| 陈怀远 | 建筑结构测绘人员 | 观察封门砖、裂缝、叠涩、结构支撑 | 砖砌结构、平剖面、叠涩、封门安全 | 不解释人物身份、剧情秘密 |
| 赵老倔 / 赵广田 | 考古技工 / 当地技工 | 拆砖、清理、判断现场风险、提供地方经验和口语氛围 | 清理动作、封门砖、工具、地方经验 | 不承担学术结论，不解释复杂题记 |
| 苏池 | 体质人类学队员 | 解释人骨性别、年龄、骨骼状态、迁葬可能 | 人骨、人类学、骨骼清理、年龄性别判断 | 不解释壁画风格、过道题记 |
| 附近村民 | 墓外 NPC | 提供地方传闻、墓群位置和开场氛围 | 墓外传闻、发现经过、地方记忆 | 不提供墓内专业判断 |

## 6. 场景与 NPC 任务细化

### CH01 抵达白沙

| 项目 | 内容 |
|---|---|
| 场景 | 墓外、白沙镇、村口、工地入口 |
| 核心内容 | 玩家首次抵达白沙，认识周淼、粟柏年、陈怀远、赵老倔、苏池 |
| NPC 任务 | 周淼接引；粟柏年测试观察；陈怀远、赵老倔、苏池建立角色印象 |
| 玩家任务 | 判断干河道走向，理解“先看地形”的考古习惯 |
| 线索类型 | E 环境线索、F 玩家私人线索 |
| 小游戏 | 无 |
| 注意 | 银扣、外祖父遗物属于 F 类剧情线索，不进入真实考古证据 |

### CH02 墓道初清

| 项目 | 内容 |
|---|---|
| 场景 | 墓道、墓门前平坦部 |
| 核心内容 | 墓道朝向、夯层、夯窝、白瓷片、记录训练 |
| NPC 任务 | 粟柏年说明夯层断代和记录规范；赵老倔提供现场经验 |
| 玩家任务 | 判断夯层意义，记录白瓷片位置、深度和状态 |
| 线索类型 | P0/E 墓道结构，P1 白瓷片记录，P2 夯窝细节 |
| 小游戏 | `mg_dig_match3` 可放在墓道清理或墓门前清理 |
| 注意 | 游戏中应避免让挖土消消乐变成纯消除，完成后必须产出“清理记录” |

### CH03 封门与墓门

## 7. 给各负责人使用的拆分

### 固定剧情 NPC 负责人

优先整理：

1. CH01 抵达白沙开场。
2. CH02 墓道夯层与记录训练。
3. CH03 封门砖拆除顺序与裂缝支撑。
4. CH05 题记辨读前后引导。
5. CH08 后室遗物定位前后引导。
6. 各章完成后的粟柏年收束语。

交付时每个节点必须写 `nodeId`、触发条件、出场 NPC、绑定 `clueId` 或 `miniGameId`、产出结果。

### 自由对话 NPC 负责人

优先只做粟柏年一个可自由对话 NPC。

每章需要提供：

| 章节 | currentGoal | allowedClues | forbiddenTopics |
|---|---|---|---|
| CH01 | 确认地形和队伍任务 | 环境线索、考古队角色 | 不谈墓内结论 |
| CH02 | 学会墓道清理和记录 | 墓道、夯层、白瓷片 | 不提前说后室地券 |
| CH03 | 判断封门结构和安全拆除 | 封门砖、裂缝、支撑 | 不解释暗格真相 |
| CH04 | 观察甬道结构和彩画 | 版门、门钉、叠涩、叠胜 | 不提前给前室结论 |
| CH05 | 判断礼仪空间和题记 | 女乐、宴饮、题记 | 不提前给后室结论 |
| CH06 | 观察妇人启门和后室入口 | 妇人启门、假门、门缝 | 不确认虚构暗格为考古事实 |
| CH08 | 汇总后室遗存关系 | 地券、人骨、铁钉、分布图 | 不把迁葬写成确定结论 |

### 小游戏负责人

| 负责人 | 小游戏 | 需要对接章节 |
|---|---|---|
| 小游戏 A | 挖土消消乐 | CH02 墓道初清 / 墓门前清理 |
| 小游戏 B | 题记辨读 | CH05 前室末尾 / 过道题记 |
| 小游戏 B 或 A/B 协作 | 后室遗物定位 | CH08 后室清理与封存 |

每个小游戏必须输出完成记录，供场景推进或记录夹使用。

### 图片建模 + 线索负责人

优先补齐以下线索和图片对应关系：

1. 墓外 / 墓道：墓道朝向、夯层、白瓷片、封门前平坦部。
2. 墓门：三层封门砖、封门砖正中拓片、裂缝、支撑。
3. 甬道：版门、门钉、门环、断砌造、顶部叠涩、叠胜彩画。
4. 前室：顶部藻井、东壁女乐、西壁宴饮、南壁侍卫与贡纳、过道题记。
5. 后室：妇人启门、地券、人骨、铁钉、镇墓钱、出土物分布图。
6. F 类剧情线索：银扣、永安、暗格、铜筒、素帛、绢帛、屏风藏字。

## 8. 当前风险与处理建议

| 风险 | 表现 | 建议 |
|---|---|---|
| 剧情虚构与考古事实混淆 | 暗格、绢帛、屏风藏字被当作真实线索 | 全部标为 F 类剧情线索 |
| 小游戏过度集中 | 前期小游戏多，后期只剩点击 | 保留前中后三段小游戏 |
| NPC 说得太满 | 直接给出结论或剧透 | 固定剧情用节点表，自由对话用禁止透露表 |
| 后室信息过重 | 人骨、地券、遗物、日常器物同时出现 | 用后室遗物定位小游戏做收束 |
| 场景与剧情脱节 | 剧情写得好但不知道放哪张图 | 每个剧情节点必须绑定 sceneId / viewId / clueId |

## 9. 建议下一步交付

下一步最适合让团队分别填写四张表：

3. 小游戏接入表：挖土消消乐、题记辨读、后室遗物定位。
4. 图片与线索对应表：为每个剧情节点补 imageId 和 clueId。
