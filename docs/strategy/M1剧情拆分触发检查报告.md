# M1 剧情拆分触发检查报告

检查依据：[M1剧情拆分触发方案.md](./M1剧情拆分触发方案.md)

## 当前结论

当前不是“完全没有 merge 成功”。底层触发系统已经接入，且主线状态机可以按以下顺序跑通：

`environment -> tomb_gate -> corridor -> front_chamber -> passage -> rear_chamber -> final_report`

但此前玩家不容易感受到剧情，主要有两个原因：

1. 已接入的是摘要型 NPC 提示，不是完整剧情文案里的强叙事版本。
2. 旧存档中已经收录过的线索、已完成的小游戏和章节不会自动重弹首次剧情，导致新增剧情被存档状态“吞掉”。

本轮已补救：

- 增加 `storyVersion` 迁移，旧存档会补播剧情系统更新提示、当前章节入场、最近章节收束、最近小游戏反馈。
- 墓外进入墓门前新增 `environment` 收束剧情。
- opening 增加林砚秋、银扣、粟柏年的开场语气。
- 后室收束增加暗格/赵怀诚 F 线提示，明确其只能作为可推测/仍存疑材料。

## 下一步建议

如果要让玩家明显感受到“剧情向内容”，下一步不应该继续只补 NPC 摘要，而应该做三件事：

1. 把 `剧情文案/加入线索后的最终剧情_融合版.md` 中的楔子、墓外、暗格、封存段落拆成可播放的剧情卡。
2. 增加终章三栏报告交互：`可判断 / 可推测 / 仍存疑`。
3. 增加暗格/F 线入口：后室完成后弹出，但明确不作为学术结论。
