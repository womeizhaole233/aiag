# M1 剧情拆分与触发方案

## 主顺序

不要把整段剧情按“大章”硬塞到每关结尾，而是拆成三类触发点：

1. **入场短剧情**：刚进入某个墓室/空间时，只给方位、角色提示、观察任务，不讲结论。
2. **线索触发剧情**：玩家点到某个关键线索后，弹一小段 NPC 反应或追问。
3. **章节收束剧情**：材料搜集 + 小游戏 + 线索整理完成后，播放本章主要剧情；剧情结束后才开放下一关。

主流程定为：

**进入章节 -> 入场短剧情 -> 搜集材料 -> 触发必要小游戏 -> 线索搜集/整理完成 -> 章节主剧情 -> 解锁下一关**

## 剧情拆分方案

| 剧情内容 | 推荐触发 | 合入位置 | 说明 |
|---|---|---|---|
| 楔子：林砚秋到白沙、银扣、人物初见 | 游戏开场前/墓外首次进入 | `environment` 前置 opening | 适合做开场，不算墓室关卡，不阻塞玩家太久 |
| 墓外：看墓群位置、M1 空间序列、外祖父线索 | 墓外材料搜集后 | `environment` 章节收束剧情 | 玩家先完成定位/空间序列，再引出“墓道->墓门->甬道->前室->过道->后室” |
| 墓门：墓道夯层、封门砖拆除顺序 | 墓门入场 + 消消乐前后 | `tomb_gate` | 入场说“先看封门结构”；清理小游戏后再讲封门砖和墓门显形 |
| 墓门：门额正背、背面彩画、“正面是规矩，背面是自由” | 墓门关键线索收齐后 | `tomb_gate` 章节收束 | 这段是本章方法论，应放在线索搜集完后，不适合一进门就讲 |
| 甬道：断砌造、门钉、顶部/两壁三面合读 | 甬道材料搜集后 | `corridor` 章节收束 | 入场只提示“抬头、看两壁”；三面合读结论等玩家点完顶部、东壁、西壁再说 |
| 甬道：为什么三面合读的选择题 | 章节剧情中 | `corridor` 剧情选择节点 | 适合做章节收束里的互动问答 |
| 前室：进入前室、盝顶藻井、陈怀远测量 | 前室首次进入 | `front_chamber` 入场短剧情 | 这段是空间氛围和任务提示，适合刚进前室 |
| 前室：不能只看漂亮画面，要人物、器物、入口、顶部同表复查 | 前室线索整理完成后 | `front_chamber` 章节收束 | 这是前室结论，应放在完成壁面/器物/顶部复查之后 |
| 过道：空间变窄、题记、窗棂、顶部结构 | 过道首次进入 | `passage` 入场短剧情 | 适合做过道入场氛围 |
| 过道：“赵大翁是谁”、题记只能作时间锚点 | 题记辨读小游戏完成后 | `passage` 线索触发 + 章节收束 | 题记小游戏后弹，强调“1099 年和称谓，不等于身份定论” |
| 后室：空间收束、六角平面、砖床基准 | 后室首次进入 | `rear_chamber` 入场短剧情 | 适合刚进入后室，让玩家知道本章重点是砖床、地券、人骨、器物 |
| 后室：铁钉、人骨、地券、日常器物分类 | 后室遗物定位小游戏和核心线索完成后 | `rear_chamber` 章节收束 | 放在后室材料全收齐后，作为后室主剧情 |
| 暗格、反切、赵怀诚线 | 后室章节完成后、终章前 | `rear_chamber` 后置剧情 / F 线 | 这段很有剧情张力，但必须标为 F 线、可推测/存疑，不能作为后室学术结论 |
| 终章：三栏报告，可判断/可推测/仍存疑 | 五章完成后 | `final_report` / 结论墙 | 这是最终玩法核心，适合做终章剧情和交互 |
| 封存/尾声 | 终章完成后 | ending | 不参与关卡解锁，只做情绪收束 |

## 每关更细的触发

| 关卡 | 入场剧情 | 中途线索剧情 | 小游戏后剧情 | 章节主剧情 / 解锁 |
|---|---|---|---|---|
| 墓外 | 到白沙、墓群位置、粟柏年让玩家先看大局 | 标记墓群范围时触发“待探/待核”提示 | 可接消消乐前的清理说明 | 完成空间序列后，讲外祖父路线和进入墓门 |
| 墓门 | 墓道夯层、封门砖、先看结构 | 点门额背面彩画、封门砖组织时弹短剧情 | 消消乐完成后，墓门显形 | 门额正背和封门证据收齐后，粟柏年总结“背面也有信息”，再进甬道 |
| 甬道 | 刚进甬道，提示“别只往前走，抬头看顶部，侧看两壁” | 点顶部叠胜纹、东壁、西壁时触发三面合读提示 | 叠胜纹小游戏完成后反馈纹样关系 | 三面材料齐后，播放“三面合读”问答，再进前室 |
| 前室 | 进入前室，陈怀远测量盝顶，提示分墙观察 | 点女乐、器物、顶部、南北壁时弹短反馈 | 暂无必做小游戏 | 完成墙面/器物/顶部复查后，播放“不能只看漂亮画面”，再进过道 |
| 过道 | 空间变窄，周淼提示题记位置 | 点题记/破子棂窗/顶部时弹短反馈 | 题记辨读完成后，讲“赵大翁”只能做时间锚点 | 过道核心线索齐后，播放题记边界剧情，再进后室 |
| 后室 | 进入后室，先看砖床和六角空间 | 点妇人启门、地券、人骨、铁钉、器物时分段触发 | 遗物定位完成后，讲遗存叠合边界 | 核心证据 + 日常器物线完成后，播放后室收束，再进入暗格/F线或终章 |
| 终章 | 粟柏年铺开六章结论卡 | 三栏报告每栏选择后反馈 | 无 | 三栏完成后播放最终收束和封存尾声 |

## 实现口径

- 入场剧情只放 1-2 个弹窗，避免打断探索。
- 线索触发剧情只绑定关键线索，不是每个热点都弹。
- 真正长剧情集中在章节完成后。
- 下一关解锁条件改为：`材料/小游戏/整理完成` 后先触发 `chapterStory`，玩家点完剧情最后一段，才执行 `navigateTo(nextScene)`。
- 暗格和赵怀诚线不直接解锁学术结论，只进入 F 线和终章三栏报告。

## 检查口径

这个方案的验收重点不是“有 NPC 弹窗”本身，而是玩家能清楚感受到：

- 刚进入章节时知道自己要看什么。
- 点到关键线索时有角色追问或方法提醒。
- 小游戏完成后剧情承认玩家刚刚做的动作。
- 整理台完成后有本章主剧情收束。
- 章节主剧情关闭后才进入下一关。
- 赵怀诚/暗格线被明确标为 F 线、可推测或仍存疑。
